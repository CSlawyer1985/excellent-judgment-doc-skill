# 法官撰写心得检查清单

> **版本**: v3.0
> **创建日期**: 2026年2月8日
> **来源**: 基于《双百优秀裁判文书》115个案例的法官撰写心得提炼
> **功能**: 帮助法官自查文书撰写质量，从"怎么做"到"怎么做好"

---

## 使用说明

**检查方式**:
- ✅ 已做到
- ⚠️ 部分做到
- ❌ 未做到

**评分标准**:
- 优秀（90-100分）：全为✅
- 良好（75-89分）：大部分✅，个别⚠️
- 合格（60-74分）：✅和⚠️各半
- 不合格（<60分）：多个❌

## 一、责任心层面检查

### 1.1 思想重视检查

#### 检查项

- [ ] **是否认真对待每一份文书？**
  - 是否将每一份文书都当成作品来打磨？
  - 是否没有因为案件简单而草率处理？
  - 是否没有因为时间紧张而降低标准？

- [ ] **是否认真审查了每个事实认定？**
  - 每个事实认定是否有证据支持？
  - 是否认真审查了证据的真实性、合法性、关联性？
  - 是否对争议事实进行了充分论证？

- [ ] **是否仔细推敲了每句话理？**
  - 每句说理是否都有法律依据？
  - 说理是否逻辑严密、无懈可击？
  - 说理是否让当事人理解并接受？

#### 自我反思

```
我是否曾经因为以下原因降低标准？
□ 案件简单，快速处理即可
□ 时间紧张，先完成再说
□ 小额案件，无需详细说理
□ 类似案件，套用模板即可

如果选择了任何一项，说明思想重视程度有待提高。
```

---

### 1.2 政治担当检查

#### 检查项

- [ ] **是否考虑了文书的社会效果？**
  - 是否考虑了裁判对社会价值观的引导？
  - 是否考虑了裁判对行业规范的影响？
  - 是否考虑了裁判对公众法治意识的影响？

- [ ] **是否融入了社会主义核心价值观？**
  - 是否在说理中体现了公正、法治、诚信、友善等价值？
  - 是否平衡了法律效果与社会效果？
  - 是否传递了司法温度？

- [ ] **是否体现了司法温度？**
  - 是否在文书中体现了对当事人的理解和关怀？
  - 是否在适用法律时考虑了当事人的实际困难？
  - 是否在裁判结果上体现了公平正义？

#### 自我反思

```
我是否只考虑了以下因素而忽视社会效果？
□ 只考虑法律条文，忽视社会影响
□ 只考虑个案公正，忽视社会导向
□ 只考虑逻辑严密，忽视人文关怀

如果选择了任何一项，说明政治担当意识有待加强。
```

---

### 1.3 价值体现检查

#### 检查项

- [ ] **裁判结果是否公平公正？**
  - 裁判结果是否符合法律规定？
  - 裁判结果是否与案件事实相适应？
  - 裁判结果是否让当事人心服口服？

- [ ] **说理是否充分？**
  - 是否对争议焦点进行了充分说理？
  - 是否对法律适用进行了充分论证？
  - 是否对裁判结果进行了充分说明？

- [ ] **当事人是否理解裁判理由？**
  - 说理是否清晰易懂？
  - 说理是否让胜诉方明白为什么胜诉？
  - 说理是否让败诉方明白为什么败诉？

#### 自我反思

```
我是否认为以下观点是正确的？
□ 当事人理解与否不重要，裁判公正即可
□ 说理充分即可，不需要考虑可理解性
□ 胜败结果最重要，说理可以简化

如果选择了任何一项，说明对价值体现的理解有待深化。
```

## 二、能力建设层面检查

### 2.1 防止内卷化检查

#### 检查项

- [ ] **是否持续学习新法规？**
  - 是否学习了最新的法律法规和司法解释？
  - 是否更新了知识体系？
  - 是否尝试了新的说理方法？

- [ ] **是否检索了类案？**
  - 是否检索了相关案例？
  - 是否参考了指导性案例？
  - 是否咨询了资深法官？

- [ ] **是否尝试新方法？**
  - 是否尝试了新的说理技巧？
  - 是否尝试了新的文书格式？
  - 是否尝试了新的论证方法？

#### 自我反思

```
我是否存在以下内卷化现象？
□ 依赖经验，忽视新法规
□ 固守旧方法，拒绝新思路
□ 重复使用模板，缺乏个性化

如果选择了任何一项，说明存在能力内卷化风险。
```

### 2.2 系统学习检查

#### 检查项

- [ ] **是否理解立法本意？**
  - 是否理解了法律条文的立法目的？
  - 是否掌握了法律的整体框架？
  - 是否准确适用了法律条文？

- [ ] **是否掌握整体框架？**
  - 是否理解了法律体系的整体架构？
  - 是否掌握了相关法律的内在联系？
  - 是否能够准确适用相关法律？

- [ ] **是否准确适用条文？**
  - 引用的法条是否准确无误？
  - 法条适用是否与案件事实相符？
  - 是否避免了断章取义？

#### 自我反思

```
我是否只关注以下方面而忽视系统性学习？
□ 只看条文，不看理论
□ 只看结果，不看过程
□ 只看表面，不看本质

如果选择了任何一项，说明系统学习有待加强。
```

### 2.3 类案检索检查

#### 检查项

- [ ] **是否检索了类案？**
  - 是否使用了中国裁判文书网？
  - 是否参考了指导性案例？
  - 是否查询了司法解释和公报案例？

- [ ] **是否遵循检索三原则？**
  - 是否优先参考高级别法院的判决？
  - 是否优先参考二审和再审判决？
  - 是否优先参考改判判决？

- [ ] **是否咨询了资深法官？**
  - 是否就疑难问题咨询了资深法官？
  - 是否参加了法官会议专题讨论？
  - 是否听取了多方意见？

#### 自我反思

```
我是否认为以下观点是正确的？
□ 自己经验足够，无需检索类案
□ 检索浪费时间，直接上手即可
□ 类案不可比，检索无意义

如果选择了任何一项，说明对类案检索的认识有待提高。
```

## 三、写作技巧层面检查

### 3.1 措辞适当检查

#### 检查项

##### 事实认定部分

- [ ] **是否客观中立？**
  - 是否不带情绪色彩？
  - 是否避免了主观臆断？
  - 是否准确描述事实？

- [ ] **是否准确规范？**
  - 是否避免了模糊表述？
  - 是否使用了法言法语？
  - 是否避免了口语化表达？

##### 说理部分

- [ ] **是否法理清晰？**
  - 说理是否逻辑严密？
  - 说理是否有理有据？
  - 说理是否简练不拖沓？

##### 文尾部分（如需要）

- [ ] **是否体现人文关怀？**
  - 是否在需要时体现了司法温度？
  - 是否在需要时引导社会风尚？
  - 是否在需要时传递正向价值？

#### 自我反思

```
我是否使用了以下不当表述？
□ 使用情绪化语言
□ 使用口语化表述
□ 过于冷漠或过于热情

如果选择了任何一项，说明措辞适当性有待改进。
```

---

### 3.2 结构优化检查

#### 检查项

##### 首部规范

- [ ] **当事人信息是否完整准确？**
  - 当事人姓名/名称是否准确？
  - 当事人信息是否齐全？
  - 当事人身份是否明确？

- [ ] **案件由来是否清晰明确？**
  - 案件由来是否清晰？
  - 程序事项是否全面记载？
  - 时间节点是否准确？

##### 正文结构

- [ ] **诉辩意见是否全面归纳？**
  - 原告诉称是否全面？
  - 被告辩称是否全面？
  - 第三人意见是否全面？

- [ ] **争议焦点是否准确提炼？**
  - 争议焦点是否完整？
  - 争议焦点是否准确？
  - 争议焦点是否层次分明？

- [ ] **证据分析是否详细充分？**
  - 证据分析是否详细？
  - 证据认证是否充分？
  - 证据链条是否完整？

- [ ] **事实认定是否清楚明白？**
  - 事实认定是否清楚？
  - 事实认定有据可依？
  - 事实认定逻辑严密？

- [ ] **裁判理由是否严密透彻？**
  - 裁判理由是否严密？
  - 裁判理由是否透彻？
  - 裁判理由是否充分？

- [ ] **裁判结果是否明确具体？**
  - 裁判结果是否明确？
  - 裁判结果是否具体？
  - 裁判结果是否可执行？

##### 尾部规范

- [ ] **诉讼费用负担是否明确？**
  - 诉讼费用是否明确？
  - 费担负担是否合理？
  - 费担计算是否准确？

- [ ] **上诉权利告知是否完整？**
  - 上诉期间是否明确？
  - 上诉法院是否明确？
  - 上诉权利是否告知？

- [ ] **署名签章是否规范？**
  - 署名是否规范？
  - 日期是否准确？
  - 签章是否齐全？

#### 自我反思

```
我是否存在以下结构问题？
□ 要素遗漏，结构残缺
□ 层次混乱，逻辑不清
□ 重点不突出，详略不当

如果选择了任何一项，说明结构优化有待改进。
```

---

### 3.3 逻辑严密检查

#### 检查项

- [ ] **是否吃透案情？**
  - 是否全面梳理了案件事实？
  - 是否准确把握了法律关系？
  - 是否识别了核心争议？

- [ ] **焦点提炼是否准确？**
  - 归纳的关键争议点是否完整？
  - 是否区分了事实与法律争议？
  - 焦点提炼是否准确无误？

- [ ] **推理是否严密？**
  - 前提是否真实可靠？
  - 推理过程是否完整？
  - 结论是否自然得出？

#### 自我反思

```
我是否在以下方面存在不足？
□ 理解肤浅，未深入
□ 焦点遗漏或模糊
□ 推理跳跃或矛盾

如果选择了任何一项，说明逻辑严密性有待加强。
```

## 四、特殊案件专项检查

- [ ] **Step 2: 凝练焦点，抓牢核心争议**
  - 是否提炼了关键事实？
  - 是否识别了核心争议？
  - 是否明确了法律关系？

- [ ] **Step 3: 直面问题，深入分析说理**
  - 是否不回避难点问题？
  - 是否深入分析说理？
  - 是否充分论证理由？

- [ ] **Step 4: 情理交融，体现法律温度**
  - 是否理解当事人处境？
  - 是否体现人文关怀？
  - 是否传递司法温度？

- [ ] **Step 5: 以案说法，彰显社会主义核心价值观**
  - 是否释法说理？
  - 是否引领风尚？
  - 是否传播法治精神？

### 5.2 改进建议

#### 如果整体质量为"优秀"

**继续保持**：
- ✅ 继续保持高标准
- ✅ 继续创新方法
- ✅ 继续培育精品文书
- ✅ 分享经验，指导他人

#### 如果整体质量为"良好"

**提升方向**：
- ⚠️ 针对薄弱环节加强
- ⚠️ 向优秀文书学习
- ⚠️ 参加培训提升能力
- ⚠️ 多写多练多反思

#### 如果整体质量为"合格"

**改进重点**：
- ❌ 系统学习写作方法
- ❌ 对照优秀文书找差距
- ❌ 请资深法官指导
- ❌ 参加专项培训

#### 如果整体质量为"不合格"

**紧急改进**：
- ❌❌ 立即停止按现有方法撰写
- ❌❌ 系统学习基础写作规范
- ❌❌ 请资深法官一对一指导
- ❌❌ 参加系统培训

### 5.3 持续改进计划

#### 短期计划（1-3个月）

**目标**: 从合格提升到良好

**行动**:
1. 每周阅读2篇优秀裁判文书
2. 每月撰写4篇文书并请资深法官点评
3. 对照检查清单逐项改进
4. 参加院内培训课程

#### 中期计划（3-6个月）

**目标**: 从良好提升到优秀

**行动**:
1. 深入学习法官撰写心得精华库
2. 掌握50个说理技巧
3. 尝试撰写精品文书
4. 总结个人裁判经验

#### 长期计划（6个月以上）

**目标**: 成为优秀法官，培育精品文书

**行动**:
1. 形成个人裁判风格
2. 培育指导性案例
3. 撰写理论研究文章
4. 指导年轻法官成长

# 民事判决书模板

## 标准模板（一审普通程序）

```
XX省XX市XX区人民法院
民事判决书

（20XX）X法民初字第XX号

原告：XXX，男/女，XXXX年XX月XX日出生，X族，工作单位：XXXX，住所地：XXXXXX。
法定代表人/主要负责人：XXX，职务：XXX。
委托诉讼代理人：XXX，XXX律师事务所律师/法律服务工作者。
委托诉讼代理人：XXX，XXX律师事务所律师/法律服务工作者。

被告：XXX，男/女，XXXX年XX月XX日出生，X族，工作单位：XXXX，住所地：XXXXXX。
法定代表人/主要负责人：XXX，职务：XXX。
委托诉讼代理人：XXX，XXX律师事务所律师/法律服务工作者。
委托诉讼代理人：XXX，XXX律师事务所律师/法律服务工作者。

第三人：XXX，男/女，XXXX年XX月XX日出生，X族，工作单位：XXXX，住所地：XXXXXX。
（以上信息根据实际情况填写）

原告XXX与被告XXX、第三人XXX（案由）纠纷一案，本院于20XX年XX月XX日立案后，依法适用普通程序/简易程序，公开/因涉及...不公开开庭进行了审理。原告XXX、被告XXX、第三人XXX（到庭情况）到庭参加诉讼。本案现已审理终结。

原告XXX向本院提出诉讼请求：1. XXX；2. XXX；3. XXX。事实和理由：...（概括原告主张的事实和理由）。

被告XXX辩称，...（概括被告的答辩意见）。

本案当事人围绕诉讼请求依法提交了证据，本院组织当事人进行了证据交换和质证。对当事人无异议的证据，本院予以确认并在卷佐证。对有争议的证据和事实，本院认定如下：

XXXX年XX月XX日，...（按时间顺序叙述案件事实）
...

本院认为，本案的争议焦点为：一、XX；二、XX；三、XX。

关于争议焦点一，...（说理：法律适用 + 事实认定 + 裁判结论）
关于争议焦点二，...
关于争议焦点三，...

综上所述，依照《中华人民共和国XX法》第XX条、《中华人民共和国XX法》第XX条、《最高人民法院关于XX的解释》第XX条之规定，判决如下：

一、被告XX于本判决生效之日起XX日内...；
二、被告XX于本判决生效之日起XX日内...；
三、...；
四、驳回原告XX的其他诉讼请求。

如果未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百六十条规定，加倍支付迟延履行期间的债务利息。

案件受理费XX元，由原告XX负担XX元，被告XX负担XX元（于本判决生效之日起XX日内向原告XX支付/向本院缴纳）。

如不服本判决，可以在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人或者代表人的人数提出副本，上诉于XX省XX市中级人民法院。

审  判  长    XXX
审  判  员    XXX
审  判  员    XXX

（院印）
XXXX年XX月XX日
```

## 关键部分填写指南

### 一、当事人基本信息

**自然人**：
- 姓名、性别、出生日期（具体到日）
- 民族、工作单位（如有）
- 住所地（户籍所在地）或经常居住地

**法人或其他组织**：
- 名称、住所地
- 法定代表人或主要负责人姓名、职务
- 统一社会信用代码（如需）

### 二、诉讼请求部分

要求：
- 具体、明确、可执行
- 符合诉讼请求的法定类型（给付、确认、形成）
- 按重要性排序

常见类型：
1. 给付之诉：被告支付XX元、被告返还XX物
2. 确认之诉：确认合同有效/无效
3. 形成之诉：解除合同、撤销决议

### 三、事实和理由部分

要求：
- 简明扼要
- 围绕诉讼请求
- 不展开论述

### 四、答辩意见部分

要求：
- 准确概括被告答辩要点
- 不遗漏重要抗辩理由
- 保持客观中立

### 五、证据部分

要求：
- 全面列举证据
- 逐项分析三性（真实性、合法性、关联性）
- 说明认证意见
- 综合判断

格式：
```
证据X：XX（当事人）提供的XX证据。证明：XX（证明目的）。
XX（对方当事人）质证认为：XX（质证意见）。
本院认为：XX（认证意见）。
```

### 六、事实认定部分

要求：
- 基于证据认定
- 要素齐全（时间、地点、主体、事件、结果、原因）
- 层次分明
- 繁简得当

格式：
```
XXXX年XX月XX日，XX（主体）在XX（地点）XX（行为），XX（结果）。
```

### 七、争议焦点归纳

要求：
- 聚焦化（通常3-5个）
- 条理化
- 具体化

常见表述：
```
本案的争议焦点为：一、XX；二、XX；三、XX。
```

### 八、裁判理由部分

要求：
- 针对争议焦点
- 事实清楚
- 法律适用准确
- 说理充分
- 逻辑严密

结构：
```
关于争议焦点X，XXX（原告主张）。
XXX（被告抗辩）。
本院认为，XXX（说理：法律 + 事实 + 结论）。
```

### 九、裁判结果部分

要求：
- 具体明确
- 可执行
- 完整无遗漏
- 与理由对应

格式：
```
一、被告XX于本判决生效之日起XX日内偿还原告XX借款本金XX元及利息
（以XX元为基数，自XX年XX月XX日起至实际清偿之日止，按照XX利率计算）；
二、...
三、驳回原告XX的其他诉讼请求。
```

### 十、尾部部分

包括：
- 诉讼费用负担
- 上诉权利告知
- 审判组织、署名
- 日期、院印

## 常见案由专用模板

### 民间借贷纠纷

```
（事实认定部分）
XXXX年XX月XX日，被告XX因XX需要向原告借款，双方签订《借款合同》，
约定借款金额XX元，借款期限XX月，年利率XX%。同日，原告通过XX方式
向被告支付XX元。借款期限届满后，被告未按约定偿还借款本息。

（争议焦点）
一、原被告之间的借贷关系是否成立；
二、被告是否应当承担还款责任；
三、借款利息应如何计算。

（裁判理由）
关于争议焦点一，《中华人民共和国民法典》第六百六十七条规定，
借款合同是借款人向贷款人借款，到期返还借款并支付利息的合同。
第六百六十八条规定，借款合同应当采用书面形式，但是自然人之间
借款另有约定的除外。本案中，原被告签订《借款合同》，原告通过
银行转账向被告支付XX元，双方形成民间借贷法律关系。

关于争议焦点二，《中华人民共和国民法典》第六百七十五条规定，
借款人应当按照约定的期限返还借款。本案中，被告未按约定偿还
借款，构成违约，依法应承担还款责任。

关于争议焦点三，《中华人民共和国民法典》第六百七十六条
规定，借款人未按照约定的期限返还借款的，应当按照约定或者
国家有关规定支付逾期利息。...

（裁判结果）
一、被告XX于本判决生效之日起XX日内偿还原告XX借款本金XX元
及利息（以XX元为基数，自XX年XX月XX日起至实际清偿之日止，
按照XX利率计算）；
```

### 买卖合同纠纷

```
（事实认定部分）
XXXX年XX月XX日，原告与被告签订《XX买卖合同》，约定被告向原告
购买XX，价款XX元，交付时间XX，付款方式XX。XXXX年XX月XX日，
原告按约定向被告交付XX，但被告未按约定支付货款。

（争议焦点）
一、原被告之间的买卖合同是否有效；
二、被告是否构成违约；
三、违约责任应如何承担。

（裁判理由）
关于争议焦点一，《中华人民共和国民法典》第五百零二条规定，
依法成立的合同，自成立时生效，但是法律另有规定或者当事人
另有约定的除外。本案中，原被告签订的《XX买卖合同》系双方
真实意思表示，内容不违反法律、行政法规的强制性规定，
合法有效。

关于争议焦点二，《中华人民共和国民法典》第六百零一条规定，
买受人应当按照约定的时间支付价款。本案中，被告未按约定
支付货款，构成违约。

关于争议焦点三，《中华人民共和国民法典》第五百七十七条规定，
当事人一方不履行合同义务或者履行合同义务不符合约定的，
应当承担继续履行、采取补救措施或者赔偿损失等违约责任。
...

（裁判结果）
一、被告XX于本判决生效之日起XX日内支付原告XX货款XX元及
违约金（以XX元为基数，自XX年XX月XX日起至实际支付之日止，
按照XX计算）；
```

### 离婚纠纷

```
（事实认定部分）
XXXX年XX月XX日，原告XX与被告XX在XX民政局登记结婚，婚后
生育一子/女XX（XX年XX月XX日出生）。现原告以夫妻感情
确已破裂为由起诉离婚。

（争议焦点）
一、夫妻感情是否确已破裂；
二、如离婚，子女由谁抚养更为适宜；
三、如离婚，财产如何分割；
四、如离婚，是否需要支付经济帮助/补偿。

（裁判理由）
关于争议焦点一，《中华人民共和国民法典》第一千零七十九条
规定，夫妻一方要求离婚的，可以由有关组织进行调解或者直接
向人民法院提起离婚诉讼。人民法院审理离婚案件，应当进行调解；
如果感情确已破裂，调解无效的，应当准予离婚。有下列情形之一，
调解无效的，应当准予离婚：（一）重婚；（二）与他人同居；
（三）实施家庭暴力；（四）虐待、遗弃家庭成员；（五）有其他
导致夫妻感情破裂的情形。本案中，...

关于争议焦点二，《中华人民共和国民法典》第一千零八十四条
规定，父母与子女间的关系，不因父母离婚而消除。离婚后，
子女无论由父或者母直接抚养，仍是父母双方的子女。离婚后，
父母对于子女仍有抚养、教育、保护的权利和义务。不满两周岁的
子女，以由母亲直接抚养为原则。已满两周岁的子女，父母双方对
抚养问题协议不成的，由人民法院根据双方的具体情况，按照最
有利于未成年子女的原则判决。子女已满八周岁的，应当尊重其
真实意愿。本案中，...

（裁判结果）
一、准予原告XX与被告XX离婚；
二、婚生子/女XX由原告XX抚养，被告XX于本判决生效之日起
每月XX日支付抚养费XX元，直至XX年满十八周岁止；
三、...
```

## 注意事项

### 1. 格式规范
- 使用法言法语
- 语气中立客观
- 避免口语化
- 标点符号正确

### 2. 内容完整
- 当事人信息完整
- 事实认定清楚
- 证据分析充分
- 法律适用准确
- 裁判结果明确

### 3. 逻辑严密
- 大小前提清晰
- 推理链条完整
- 结论自然导出
- 无自相矛盾

### 4. 说理充分
- 回应争议焦点
- 证据分析深入
- 法律适用有据
- 价值判断正当

# 优秀裁判文书 Skill 快速参考指南

## 一分钟速览

优秀裁判文书 Skill 用于：
- **生成**高质量裁判文书
- **评判**裁判文书质量
- **优化**现有裁判文书

---

## 核心原则

### 三大原则
1. **以事实为根据，以法律为准绳**
2. **程序公正与实体公正并重**
3. **法律效果与社会效果统一**

### 裁判思维五要素
- 法律思维
- 逻辑思维
- 证据思维
- 价值衡量思维
- 实践思维

---

## 裁判文书质量标准

### 形式标准（10%）
- 格式规范
- 结构完整
- 要素齐全

### 实质标准（55%）
- 事实清楚（20%）
- 法律正确（25%）
- 说理充分（25%）

### 质量标准（25%）
- 结果明确（10%）
- 语言精练（10%）
- 逻辑严密（5%，计入说理）

### 评分标准
- 优秀：90分以上
- 良好：75-89分
- 合格：60-74分
- 不合格：60分以下

## 常见错误速查

| 类型 | 错误示例 | 正确示例 |
|-----|---------|---------|
| 事实 | 被告向原告借款 | 原告提供了借条、转账记录，证明被告向原告借款 |
| 法律 | 根据合同法 | 根据《中华人民共和国民法典》第XX条 |
| 说理 | 原告请求，予以支持 | 原告请求有事实和法律依据：一、...；二、... |
| 语言 | 签合同 | 订立合同 |
| 结果 | 被告承担责任 | 被告于本判决生效之日起10日内偿还原告借款100万元 |

## 核心法律条款速查

### 民法典核心条款
- 合同效力：第143条（民事法律行为有效）、第153条（违反强制性规定无效）
- 违约责任：第577条（违约责任方式）、第585条（违约金）
- 借款合同：第667条（定义）、第675条（还款义务）、第676条（逾期利息）
- 买卖合同：第595条（定义）、第601条（付款义务）、第626条（价款）

### 诉讼法核心条款
- 起诉条件：民诉法第122条
- 管辖：民诉法第22条（一般地域管辖）、第34条（协议管辖）
- 举证责任：民诉法第67条（谁主张谁举证）
- 上诉期：民诉法第171条（15日）

## 争议焦点归纳示例

### 民间借贷纠纷
1. 原被告之间的借贷关系是否成立
2. 被告是否应当承担还款责任
3. 借款利息应如何计算

### 买卖合同纠纷
1. 原被告之间的买卖合同是否有效
2. 被告是否构成违约
3. 违约责任应如何承担

### 离婚纠纷
1. 夫妻感情是否确已破裂
2. 如离婚，子女由谁抚养更为适宜
3. 如离婚，财产如何分割

## 证据分析标准

### 三性审查
- **真实性**：证据来源可靠、内容真实、形式完整
- **合法性**：取得手段合法、形式合法、程序合法
- **关联性**：与待证事实相关、能证明待证事实

### 证明标准
- 民事案件：高度可能性（>50%）
- 特殊情形：排除合理怀疑（如欺诈、恶意串通）
- 程序事项：盖然性

## 法律适用三段论

```
大前提：《民法典》第X条规定：...
小前提：本案中，被告...（事实）
结 论：被告应当...（裁判结果）
```

## 裁判结果表述规范

### 给付之诉
```
被告XX于本判决生效之日起XX日内：
一、偿还原告XX借款本金XX元；
二、支付利息（以XX元为基数，自XX年XX月XX日起至实际清偿
  之日止，按照XX利率计算）；
三、...；
四、驳回原告XX的其他诉讼请求。
```

### 确认之诉
```
确认原告XX与被告XX于XX年XX月XX日签订的《XX合同》有效（或无效）。
```

### 形成之诉
```
撤销被告XX于XX年XX月XX日作出的XX决定。
```

## 使用场景示例

### 场景1：生成裁判文书

**用户输入**：
```
请帮我写一份民间借贷纠纷的民事判决书。
原告：张三，男，1990年5月1日出生，汉族，住上海市XX区XX路XX号
被告：李四，男，1985年3月15日出生，汉族，住上海市XX区XX路XX号
事实：2020年6月1日，被告向原告借款100万元，约定年利率6%，
      借款期限1年。到期后被告未还款。
```

# 案例038：武汉中源电气有限公司合同纠纷案

- **案号**：(2018)鄂民终XXX号

- **关键词**：合同解除、违约责任、直接损失、可得利益
- **难度等级**：⭐⭐⭐⭐（高）
- **审理法院**：湖北省黄冈市中级人民法院

## 三、【裁判要点】
1. **可得利益的确定方法**：可得利益以违约方在订立合同时可预见或应当预见到的范围为限
2. **酌情确定数额**：人民法院在难以准确认定可得利益损失数额时，可以守约方请求赔偿的数额为基础，按照诚信原则和公平原则确定具体数额
3. **营商环境优化**：保护守约方合法权益，惩处违约行为，构建社会诚信体系
4. **民营企业保护**：依法、平等保护市场主体合法利益，助力打造法治化营商环境

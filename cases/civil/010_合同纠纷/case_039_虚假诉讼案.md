# 案例039：赵某1等虚假诉讼案

## 一、案例基本信息
- **案号**：(2018)鄂民再252号
- **案由**：合同纠纷
- **关键词**：虚假诉讼、当事人身份关系、利益冲突、虚构债务
- **难度等级**：⭐⭐⭐⭐⭐（最高）
- **审理法院**：最高人民法院
- **承办法官**：方芳

## 二、【撰写心得】
**核心观点：虚假诉讼的认定方法**
> "对于虚假诉讼的认定，在诉讼双方具有紧密关联关系的情况下，对关联当事人作出的陈述、提供的证据，应严格审查并结合其他证据审慎作出判断。"

**四个审查重点**：
1. **重视审查当事人的身份关系**：关注当事人之间是否具有关联关系
2. **严格审查关联当事人陈述和证据**：对关联当事人作出的陈述、提供的证据，应严格审查并结合其他证据审慎作出判断
3. **重点审查诉讼双方是否存在真实的利益冲突**：真实利益冲突是判断是否构成虚假诉讼的核心内容
4. **重点审查虚假诉讼是否侵害了案外人的合法权益**：侵害案外人合法权益是当事人进行虚假诉讼的根本目的所在

## 三、【裁判要点】
1. **虚假诉讼的识别**：公司法定代表人利用其掌管公司印章的便利条件或公司实际控制人利用其优势地位，虚构债务，试图通过诉讼获取人民法院生效判决
2. **身份关系审查**：重视审查当事人的身份关系，关注当事人之间是否具有关联关系
3. **利益冲突审查**：通过诉讼参与人的庭审表现、诉讼标的最终归属等细节，甄别诉讼双方是否存在真实的利益冲突
4. **损害其他股东权益**：通过虚假诉讼损害其他股东合法权益，应认定为虚假诉讼

# 案例047：平顶山市第五人民医院、平顶山市第二人民医院和陈某等医疗损害责任纠纷案

## 一、案例基本信息
- **案号**：(2018)豫民终XXX号
- **案由**：医疗损害责任纠纷
- **关键词**：医疗损害、诊疗行为过错、因果关系、如实告知义务、尸体解剖
- **难度等级**：⭐⭐⭐⭐⭐（最高）
- **审理法院**：河南省平顶山市中级人民法院
- **承办法官**：侯晓宏

## 二、【撰写心得】
**核心观点：医疗案件"五步法"**
1. **充分准备,全面把握案情**：制作详尽的阅卷笔录,审查关键案件证据,确认争议细节
2. **做好类案检索工作**：对非常用法律的检索和学习是避免法律适用出现问题的必要环节
3. **凝练焦点,抓牢核心争议**：固定诉辩主张,筛选核心诉求,坚持去同留异,凝练争议事项
4. **直面问题,深入分析说理**：强化说理的针对性、增强论述的逻辑性、提升论述的严谨性
5. **情理交融,体现法律温度**：法官的情感应是出于正义和良知,以情感消融矛盾的坚冰

## 三、【裁判要点】
1. **诊疗过错认定**：医疗机构在明知其自身医疗条件和专业水平不能有效治愈患者疾病的情况下,未能采取积极措施及时建议或要求患者转院治疗,属于诊疗行为过错
2. **因果关系判断**：医疗机构对患者病情严重程度认识不足导致的诊疗行为过错,应当根据该过错对患者损害结果的参与度承担相应的赔偿责任
3. **尸体解剖义务**：患者在医疗机构内经抢救无效死亡,死者亲属未对死者死亡原因提出异议,亦未要求解剖死者尸体确定死因,而将死者尸体自行处置,导致死者死因无法查明的,死者亲属要求医疗机构承担医疗过错责任,人民法院不予支持
4. **告知义务**：医疗机构就尸体解剖事宜是否负有提示告知义务,是判断责任的关键

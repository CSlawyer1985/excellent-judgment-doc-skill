# 案例053：重庆市巴岳山矿泉水有限公司和重庆华琦精细化工有限公司环境污染责任纠纷案

## 一、案例基本信息
- **案号**：(2018)渝01民终XXX号
- **案由**：环境污染责任纠纷
- **关键词**：环境侵权、水污染、因果关系认定、承压地下水、举证责任
- **难度等级**：⭐⭐⭐⭐（高）
- **审理法院**：重庆市第一中级人民法院
- **承办法官**：李立新

## 二、【撰写心得】
**五个要点**：
1. **日常审理中做到"有心"**：收案后每一环节,阅卷、庭前准备及审理工作中,我们都可多一份就案办案之外的思考,敏锐地找寻个案价值
2. **案件事实是关键**：一篇好的文书,必然源于好的庭审,这要求法官需重视审理工作
3. **加强证据审查和资料查阅工作**：在案件审理中涉及专业性较强的问题,多学习多向专家咨询请教
4. **发挥合议庭、法官会议机制作用**：合议庭的讨论不能浮于表面,除了看到案件的法律价值之外,更应当看到案件处理的社会效果
5. **精心打磨文书**：文书的构架、说理分析,与我们日常多角度深层次的学习和积累息息相关

## 三、【裁判要点】
1. **因果关系的综合认定**：鉴于地下水水质变化原因多样,可参考污染物排放时间、污染物种类与指标、地下水水质属性、水质变化情况等因素,综合认定排污行为与地下水水质变化之间是否存在因果关系
2. **承压地下水的特征**：承压地下水是充满两个隔水层之间的含水层中的地下水,其埋藏深,不会受到地表水的渗漏,其水位、水量、水质等受水文、气候因素变化影响也较小,动态相对稳定,其不易受到污染
3. **举证责任的完成**：被侵权人应当提供证明以下事实的证据材料：
   - 污染者排放了污染物
   - 被侵权人的损害
   - 污染者排放的污染物或者其次生污染物与损害之间具有关联性
4. **专业知识的运用**：法官通过咨询相关专家,了解承压地下水特征,并将专家意见运用于裁判文书说理

# 案例062：新郑市薛店镇人民政府、新郑市薛店镇花庄村民委员会和中国生物多样性保护与绿色发展基金会等环境民事公益诉讼案

## 一、案例基本信息
- **案号**：(2017)豫民终XXX号
- **案由**：环境民事公益诉讼
- **关键词**：生态保护民事公益诉讼、古树名木、非法移栽、生态服务功能
- **难度等级**：⭐⭐⭐⭐⭐(最高)
- **审理法院**：河南省高级人民法院
- **承办法官**：高光

## 二、【撰写心得】
**五个坚持**：
1. **坚持尊重事实、依法发挥职能作用**：多次去枣树砍伐和移栽地实地勘察,走访相关单位、社会组织及村民
2. **坚持注重预防原则**：判令停止继续实施非法移栽或者采伐枣树的行为,防止后续继续非法实施破坏生态的行为
3. **坚持修复为主原则**：适用异地修复、限期履行、第三方治理等生态修复责任方式
4. **坚持损害担责原则**：判令承担赔偿性责任,赔偿服务功能损失
5. **坚持法律效果和社会效果相统一原则**：200名师生旁听庭审,加强法治教育,强化高校师生的环保理念

## 三、【裁判要点】
1. **古树名木的认定**：古树名木的认定,要在相关行政机关的组织下,进行专家鉴定、定级,统一登记、编号、建档
2. **侵权行为主体的确定**：侵权责任应当由实施侵权行为的人承担
3. **破坏生态的侵权责任**：在未办理采伐手续的情况下,违法移栽枣树并导致枣树死亡,致使生态环境被破坏,应当承担破坏生态的侵权责任
4. **生态服务功能费用的确定**：参照河南省2016年平均每亩林地森林生态价值为3644.15元的标准,酌定涉案的198.5亩枣林地服务功能损失为5年的森林生态价值

# 案例055：安徽海德化工科技有限公司和江苏省人民政府环境污染侵权赔偿纠纷案

## 一、案例基本信息
- **案号**：(2018)苏民终1316号
- **案由**：环境污染侵权赔偿纠纷
- **关键词**：民事生态环境损害赔偿诉讼、分期支付、危险废物处置资质、长江污染
- **难度等级**：⭐⭐⭐⭐⭐（最高）
- **审理法院**：江苏省高级人民法院
- **承办法官**：陈迎

## 二、【撰写心得】
**核心观点：优秀的裁判文书来自千锤百炼**

**六个要点**：
1. **归纳案由应当准确**：案由归纳准确,意味着我们所选择的法律条文不会出现明显偏差
2. **归纳诉辩意见要条理清晰**：按照诉讼的基本逻辑,依照法律规定及请求权分析法的内在逻辑归纳
3. **分组列举相关证据**：根据请求权基础的内在逻辑以及举证责任的分配规则,分组列举相关证据
4. **归纳的争议焦点应当明确具体**：按照诉讼的逻辑架构予以明确,可以分不同层次、不同位阶
5. **裁判理由要符合演绎推理的基本要求**：遵循演绎推理的基本要求,给出相应的结论
6. **裁判文书的制作要尽可能地做到通俗易懂**：善于用老百姓看得懂、能明白的语言,清晰、准确地认定事实、说明理由

## 三、【裁判要点】
1. **危险废物处置义务**：企业事业单位和其他生产经营者将生产经营过程中产生的危险废物交由不具备危险废物处置资质的企业或者个人进行处置,造成环境污染的,应当承担生态环境损害责任
2. **生态环境服务功能损失**：在生态环境恢复原状之前,社会公众丧失所应享有的生态系统服务利益,应予赔偿
3. **分期支付赔偿费用**：人民法院可以综合考虑企业事业单位和其他生产经营者的主观过错、经营状况等因素,在责任人提供有效担保后判决其分期支付赔偿费用
4. **类比方法的适用**：可以采用类比的方式,参照污染物、污染区域、污染时间大致基本相同的评估结果计算生态环境损害

# 案例051：韩某某和中国石油天然气股份有限公司吉林油田分公司水污染责任纠纷案

## 一、案例基本信息
- **案号**：(2018)最高法民申XXX号
- **案由**：水污染责任纠纷
- **关键词**：环境污染责任、侵权行为构成、损失认定、责任承担、举证责任倒置
- **难度等级**：⭐⭐⭐⭐⭐（最高）
- **审理法院**：最高人民法院
- **承办法官**：魏文超

## 二、【撰写心得】
**核心观点：环境侵权行为的构成认定以及举证责任的适用**
> "环境侵权责任适用无过错归责原则,即不问行为人主观上是否有过错,只要其行为与损害后果间存在因果关系,就应承担民事责任。"

**关键要点**：
1. **环境侵权构成要件**：环境侵权行为、损害事实、环境侵权行为与损害事实之间存在因果关系三要件
2. **举证责任分配**：被侵权人应就污染者存在污染行为、自己受到损害承担举证责任,此外还应举出初步证据证明污染物或者其次生污染物与损害之间具有关联性
3. **"低度盖然性"标准**：被侵权人的证明目标是举证证明因果关系的可能性而非确定性,污染行为与损害之间具有关联性的证明标准为低度盖然性标准
4. **消极污染行为**：排放污染物行为,不限于积极地投放或导入污染物质的行为,还包括伴随企业生产活动的消极污染行为

## 三、【裁判要点】
1. **无过错责任原则**：环境侵权责任适用无过错归责原则,污染者应当承担侵权责任,不问其主观上是否有过错
2. **举证责任倒置**：污染者应当就法律规定的不承担责任或者减轻责任的情形及其行为与损害之间不存在因果关系承担举证责任
3. **被侵权人的举证责任**：
   - 污染者排放了污染物
   - 被侵权人的损害
   - 污染者排放的污染物或者其次生污染物与损害之间具有关联性
4. **不承担责任或减轻责任的情形**：不可抗力、被侵权人故意、被侵权人重大过失、第三人过错等

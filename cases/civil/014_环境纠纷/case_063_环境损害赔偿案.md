# 案例063：重庆市人民检察院第三分院和张某环境损害赔偿案

## 一、案例基本信息
- **案号**：(2018)渝03民初XXX号
- **案由**：环境损害赔偿
- **关键词**：检察公益诉讼、环境损害赔偿、专家参与、代履行
- **难度等级**：⭐⭐⭐⭐(高)
- **审理法院**：重庆市第三中级人民法院
- **承办法官**：贺付琴

## 二、【撰写心得】
**核心观点：环境修复优先原则**
> "环境公益诉讼案件确定被告应承担的民事责任时,应贯彻生态修复优先原则,在被告无力履行环境修复义务的情况下,判令其承担修复费用并承担环境自受到损害至恢复期间的环境服务功能损失。"

**四个建议**：
1. **人民法院应主导环境损害赔偿类公益诉讼案件的执行**：探索"恢复性司法实践+社会化综合治理"的工作机制
2. **建议在全国层面设立环境修复公益基金**：借鉴贵州清镇市人民法院的生态环境修复专项基金模式
3. **倡导大力发展生态环境修复第三方市场主体**：保障环境公益组织健康发展,大力发展环境修复和治理专业机构
4. **进一步发挥专家作用**：鼓励和支持专家担任环境公益诉讼案件的人民陪审员,探索专家证人制度

## 三、【裁判要点】
1. **生态修复优先原则**：在被告无力履行环境修复义务的情况下,判令其承担修复费用并承担环境服务功能损失
2. **专家参与的运用**：环境服务功能损失数额的确定可以引入专家参与诉讼,形成专家意见,法院参考专家意见结合案件实际情况进行酌定
3. **代履行方式的适用**：选择判令被告承担相应的修复费用,并在判决主文中明确该款用于当地受损生态环境修复或与其相关的公共生态环境修复,此系环境修复义务"代履行"的积极探索
4. **检察公益诉讼的主体资格**：人民检察院在履行诉前公告程序后,没有法律规定的组织提起民事公益诉讼的情况下,可以依法向人民法院提起公益诉讼

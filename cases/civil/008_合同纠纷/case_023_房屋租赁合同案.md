## 二、【撰写心得】

### 2.1 裁判文书的意义

**要点1：裁判文书是法官职业魅力最直接的体现**
> "裁判文书是法官的产品，是名片，也是法官的档案。一个庭审的生命是短暂的，但裁判文书则不同，它会长久地带着法官的名字存在于这个世界上。"

**要点2：裁判文书是公平正义实现的过程**
> "公平正义是司法的首要价值。在我们手中完成的每一份裁判文书都承载着定分止争、确立规则的使命。"

**要点3：裁判文书是一场与当事人的对话**
> "裁判文书应当是法官就这个案件要对当事人讲的全部，是写给当事人的答案。它应该完整的记载整个案件的事实过程、法律依据以及法官的推理过程。"

### 2.2 整理争议焦点的技巧

**按照法律规范的构成要件来整理争点**（规范化、程序化、可复制的方法）：

1. **固定诉讼请求、答辩意见**
2. **将诉讼请求、答辩意见与请求权、抗辩权基础法律规范相对应**
3. **分析法律规范的构成要件**（包括隐含要件）
4. **整理法律争点**（法条理解和填补、法条冲突选择等）
5. **将规范要件与案件事实对应**
6. **整理事实争点**（每一要件事实是否存在）

**图示**：
```
诉讼请求 + 答辩意见
→ 请求权、抗辩权基础规范
→ 规范构成要件
→ 对应到每一项要件事实
→ 法律争点 + 事实争点
```

## 三、【裁判要点】

1. **可预见性规则的适用**：违约方当事人在合同订立时能够预见或理应预见到可得利益损失的性质或类型，不要求能够预见到损失的具体程度或数额

2. **限制性规则的综合运用**：在守约方对于可得利益的具体数额举证不足的情况下，应当分别运用可预见性规则、减损规则、损益相抵规则对于可得利益的边界进行界定

3. **自由裁量权的规范**：确保对可得利益损失数额作出的酌定具有较为充分的依据

4. **争议焦点的归纳方法**：按照法律规范的构成要件来整理争点，可以避免遗漏争议焦点，解决归纳争议焦点过于笼统或过于琐碎的问题

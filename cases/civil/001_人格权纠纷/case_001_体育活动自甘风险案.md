# 案例001：吉某某等和殷某1等生命权健康权身体权纠纷案

## 一、案例基本信息

- **案号**：（2016）京03民终XXX号
- **案由**：生命权、身体权、健康权纠纷
- **关键词**：体育活动自甘风险、非故意侵权、减轻责任比例、民法典第1176条
- **难度等级**：⭐⭐⭐（较高）
- **审理法院**：北京市第三中级人民法院
- **承办法官**：郑吉喆

## 二、【撰写心得】（法官第一手经验）

### 2.1 背景情境

> 本案涉及未成年人学习马术过程中不幸身亡的侵权案件，是《民法典》实施前适用"自甘风险"原则的典型案例。该案涉及马术运动这一新兴领域，法律规范尚不健全，如何准确界定责任承担具有较大挑战性。

**撰写难点**：
- 如何准确把握"自甘风险"的适用条件（当时法律尚未明确规定）
- 如何区分"自甘风险"与"过错责任"的界限
- 如何合理确定减轻责任的比例
- 如何在保护未成年人权益与鼓励文体活动发展之间取得平衡

### 2.2 解决方案

> 在撰写过程中，我重点把握了以下几点：

**要点1：思想重视，以高度责任心认真对待**

- 不能因为案件标的看似不大就草率处理
- 微小裁判可能带来巨大传播效果和后续影响
- 对有争议的裁判观点要充分重视，今天的严谨就是避免明天的攻击
- 要以最大的认真、最高的谨慎慎重对待争议裁判观点的说理

**要点2：加强学习，防止能力出现内卷化效应**

- 主动适应新型法律关系（如马术培训等新兴领域）
- 遇到争议问题要弄懂学通，不可似是而非
- 争议问题是机遇而非危机，要利用好手边工具广泛搜集资料
- 具体做法：
  - 合议庭走访了体育总局
  - 参加了多个体育法有关的年会和论坛
  - 走访了体育法学界的多位专家
  - 在院领导支持下召开了与该案有关问题的研讨会
  - 形成了两种裁判思路，并公布合议庭少数意见

**要点3：类案检索，广泛借鉴前人优秀经验**

- 掌握好类案的检索方法：准确识别特色关键词、框定案由和审判级别
- 借助信息化软件，对有特点的说理及时留存，形成自己的文书说理库
- 在撰写前检索了中国裁判文书网中所有体现体育运动中自甘风险原则的案件
- 对优秀文书的说理进行消化吸收

**类案检索五原则**：
1. **高级别法院优先**：确保参照的权威性
2. **二审和再审判决优先**：确保相关裁判观点和说理经过多级法院审理
3. **改判判决优先**：改判判决往往说理质量更高
4. **文书时间尽量选择新文书**：特别对于近期出台指导意见的案例
5. **关注地区审判指导意见**：如果多个判决说理相似且地区相近，应检索该地区是否已有审判指导意见

**要点4：措辞适当，提升裁判文书的温度和品质**

- 裁判文书要明事理合常理，符合一般大众对事物的认知
- 不能陷入法律逻辑怪圈，要避免得出奇葩论述
- 语言表达不能过于激烈，要在坚持中立前提下对败诉方诉讼理由进行有理有据回应
- 在有效回应当事人诉求的前提下，强化精品意识
- 通过引入社会主义核心价值观说理，传递正确的价值观和道德导向

### 2.3 经验教训

## 四、【裁判思路】（结构化分析）

### 4.1 四元结构分析

#### 第一元：界定民事主体

- **受害人**：殷某2（未成年人，学习马术的学员）
- **侵权人**：吉某某（骑手教练，马匹的饲养人和调教人）
- **责任人**：
  - 克来务公司：马场经营者，马匹的饲养人和管理人
  - 吉某某：教练，马匹的饲养人和调教人
- **利害关系人**：殷某1、杨某某（殷某2的父母）

#### 第二元：判断法律行为

- **侵权行为**：2015年5月17日，殷某2在吉某某的指导下学习骑马，马匹受惊致使殷某2从马上摔下，导致被马匹踩踏右脑后耳根部，经抢救无效死亡
- **行为性质**：正常的马术培训活动（非故意、非重大过失）
- **因果关系**：马匹受惊 → 殷某2摔下马 → 被马匹踩踏 → 死亡结果
- **过错认定**：
  - 吉某某：具有资质证书，从事骑马运动多年，应知晓马术运动的高风险性，未尽到审慎的注意义务
  - 克来务公司：知晓骑马运动系高风险性体育运动，未采取必要安全保障措施

#### 第三元：保障民事权利

- **生命权、健康权、身体权**：殷某2的基础权利，应当优先保护（特别是未成年人）
- **文体活动自由**：参与马术运动的自由，应当予以尊重
- **权利冲突**：未成年人生命健康权保护 vs 马术运动发展
- **权利排序**：
  - 一般情况下：生命权、健康权优先
  - 特殊情况下（自甘风险）：适当平衡，减轻非故意侵权方责任

#### 第四元：划分民事责任

- **一审裁判**：克来务公司、吉某某连带赔偿1062373.18元（全额）
- **二审裁判**：
  - 认定殷某2及其父母构成自甘风险
  - 酌情减轻克来务公司、吉某某20%的赔偿比例
  - 判决克来务公司、吉某某连带赔偿849898.54元
- **责任理由**：
  - 马术运动具有天然风险性，参与者自愿参加即意味着对风险的认知和接受
  - 在非故意侵权情形下应当减轻体育活动中侵权方的民事责任
  - 自甘风险的减轻责任比例应当与该项体育活动的发展情况及运动中风险告知呈正相关趋势
  - 考虑到马术运动现阶段发展情况、相关培训中的规范及风险告知体系尚不健全，酌情减轻20%

### 4.2 三层说理结构

#### 第一层：程序说理

- **一审审理**：明确当事人诉辩主张，归纳争议焦点
- **二审审理**：
  - 合议庭多数意见：适用自甘风险原则
  - 合议庭少数意见：应适用过错责任原则
  - 最终采纳多数意见，但同时公布少数意见

#### 第二层：实体说理

**事实认定**：
- **时间链条**：
  - 2007年1月10日：殷某2出生
  - 2014年8月30日：殷某1向吉某某汇款11800元（50次学习骑马的学费）
  - 2015年1月24日：殷某1向吉某某转账20万元购马款
  - 2015年4月4日：杨某某向吉某某转账12200元（马匹饲养费和马术训练课程费）
  - 2015年5月17日：殷某2在马术培训中受伤，经抢救无效死亡
- **证据链**：银行转账记录、庭审陈述、马匹饲养事实等
- **争议焦点**：
  1. 马术运动是否属于高风险性体育运动
  2. 殷某2及其父母是否构成自甘风险
  3. 吉某某和克来务公司是否有过错
  4. 是否应当减轻责任，减轻多少

**法律适用**：
- **大前提**（当时尚无明确规定，参照法理）：
  - 自甘风险规则：自愿参加具有一定风险的文体活动，因其他参加者的行为受到损害的，受害人不得请求其他参加者承担侵权责任；但是，其他参加者对损害的发生有故意或者重大过失的除外
- **小前提**：
  - 殷某2经父母允许进行骑乘学习活动，自愿参加马术运动
  - 马术运动具有天然的风险性
  - 吉某某、克来务公司无故意或重大过失
- **涵摄**：
  - 殷某2及其父母的行为构成自甘风险
  - 吉某某、克来务公司无故意或重大过失
  - 应当适当减轻非故意侵权方的赔偿责任
- **结论**：酌情减轻克来务公司、吉某某20%的赔偿比例

#### 第三层：价值说理

- **价值识别**：
  - 个体权益保护：未成年人生命健康权的绝对保护
  - 社会活动自由：马术运动等文体活动的健康发展
  - 价值冲突：保护未成年人权益 vs 鼓励文体活动发展

- **价值排序**：
  - 第一顺位：未成年人生命健康权保护
  - 第二顺位：文体活动自由的鼓励和保护
  - 第三顺位：风险认知和风险自担原则

- **价值论证**：
  - 马术运动具有天然的竞技性和风险性
  - 参与者自愿参加即意味着对风险的认知和接受
  - 在参与者无故意或重大过失的情况下，要求其对正常竞技行为造成的损害承担全部责任，不仅违背公平原则，也会抑制公众参与文体活动的积极性
  - 自甘风险的减轻责任比例应当与该项体育活动的发展情况及运动中风险告知呈正相关趋势

- **价值结论**：
  - 适用"自甘风险"规则，适当减轻非故意侵权方20%的责任
  - 既保护了未成年人生命健康权，又鼓励了马术运动的健康发展
  - 实现了法律效果与社会效果的统一

### 4.3 五理说理结构

#### 事实理（基础）

- **时间要素**：完整的时间链条（从殷某2出生到事故发生）
- **地点要素**：克来务公司马场
- **人物要素**：
  - 殷某2：未成年人，学习马术的学员
  - 吉某某：专业骑手，马术教练

- **原因要素**：马匹受惊（非故意、非重大过失）
- **结果要素**：殷某2死亡，产生医疗费、丧葬费、死亡赔偿金、精神损害抚慰金等损失

#### 法理（标准）

- **法律条文**（当时尚无明确规定，参照法理）：
  - 自甘风险规则（后为《民法典》第1176条吸收）
- **立法本意**：
  - 鼓励文体活动发展
  - 明确风险自担原则
  - 平衡个体权益保护与社会活动自由
- **构成要件**：
  1. 自愿参加
  2. 一定风险的活动
  3. 因其他参加者行为受损
  4. 无故意或重大过失
- **法律效果**：
  - 一般情况下：不得请求其他参加者承担侵权责任
  - 本案特殊情况：适当减轻（而非免除）20%的责任
  - 减轻理由：马术运动发展情况、风险告知体系尚不健全

  - 理论基础：民事主体得自由处分自己的权利
- **比较法**：
  - 美国法：assumption of risk原则
  - 德国法：自身风险的承担
- **学理评价**：
  - 本案适用自甘风险规则，符合法理和学理
  - 考虑到未成年人保护和马术运动发展现状，采取适当减轻而非完全免除的做法，更具合理性

## 五、【说理技巧】（可复用模板）

### 5.1 技巧1：合议庭意见公开法 ⭐⭐⭐

**适用场景**：
- 疑难复杂案件
- 新型案件
- 具有规则探索价值的案件
- 需要增强裁判透明度的案件

**操作步骤**：
1. **准确归纳争议焦点**：明确案件的核心争议点
2. **展示多数意见**：充分说明合议庭多数意见的裁判思路和理由
3. **展示少数意见**：客观呈现合议庭少数意见的不同观点和理由
4. **说明最终决定**：说明最终采纳多数意见的理由和考量
5. **确保说理充分**：多数意见和少数意见都要有充分的说理

**经典段落**（可直接引用）：
```markdown
二审法院审理后，合议庭多数意见认为，本案中殷某2的损害后果系由克来务公司提供的马匹造成，克来务公司认可系受他人所托负有受托管理职责，吉某某亦认可在平时实际从事了对马匹进行饲养和调教，系涉案马匹的饲养人和调教人。所以应当将克来务公司及吉某某作为涉案马匹的共同饲养人和管理人，应当共同对殷某2的损害后果承担民事责任。

合议庭少数意见认为，本案实质系从事体育活动的未成年人在参加体育活动时不幸受到伤害，马匹应被视为马术活动中体育器械，涉案纠纷在外观上更接近于生命权纠纷，应以过错责任归责。吉某某在教学培训中对殷某2的生命安全未尽到注意义务，存在过错；克来务公司知晓骑马运动特别是未成年人从事骑马活动的高风险性，但克来务公司却放任这种风险的发生，在不具备经营马术培训的资格下，并未针对这种风险活动予以制止或者采取任何防范措施减少风险的发生。因而克来务公司对殷某2的死亡具有过错。

合议庭认为，本案应当考虑殷某2从事骑马运动的特殊因素，即自甘风险原则在体育运动人身损害中的适用。本案中殷某2经父母允许进行骑乘学习活动，应当认为殷某2及其父母殷某1、杨某某的行为构成自甘风险，应当适当减轻非故意侵权方克来务公司、吉某某的赔偿责任，结合骑马运动现阶段的发展情况、相关培训中的规范及风险告知体系尚不健全等因素，酌情减轻克来务公司、吉某某20%的赔偿比例。
```

**效果评价**：⭐⭐⭐（优秀）
- 增强裁判透明度，体现司法公开
- 展示法官理性思辨过程
- 有助于司法权威的树立

### 5.2 技巧2：价值衡量法 ⭐⭐⭐

> "（1）高级别法院优先，这是为了确保参照的权威性。（2）二审和再审判决优先，能够确保相关裁判观点和说理经过多级法院的审理，避免裁判的恣意性。（3）改判判决优先，改判判决因为涉及对原判决理由的评判及需要用更充分的理由支撑自己的裁判观点，且往往经过法官会议的讨论程序，往往说理质量更高。（4）文书的时间尽量选择新文书，特别对于近期出台过全国性指导意见的案例适用，如果时间选择过早；则结论可能不符合最新的审判指导文件精神。（5）如果发现多个判决的说理相似，如果地区相近，：则说明该地区一般可能存在着相应的审判指导意见，那么应该检索该地区是否已经有出台的指导意见据以参考。"

**摘录理由**：类案检索的系统化方法，具有实操价值

### 6.9 防止能力内卷化

> "内卷化效应是指长期从事某一方面的工作，水平稳定，不断重复，进而自我懈怠，无渐进式的增长；无突变式的发展，对即将到来的变化没有任何准备，完全缺乏应变能力。当今社会，现代科技和社会生活日新月异，新型法律关系不断涌现，传统的法律关系也在政策和科技的驱动下产生了新的特点，以电子数据等新型证据作为证据形式的案件比重不断上升，云法庭的广泛应用带来对传统诉讼程序的冲击和证据的领域新的争议，当事人基于新的规范性文件所提出的新的诉讼理由层出不穷，如果不加强学习主动适应，很快就会落伍于时代。"

**摘录理由**：对"能力内卷化"的深刻认识，具有警示意义

### 6.10 精品意识

> "在能够有效回应当事人诉求的前提下，对于事实上有新闻点及法律适用上有新意的裁判文书要强化精品意识。通过引入社会主义核心价值观说理的方式对行为细节的性质进行评判，从而传递正确的价值观和道德导向，要让公众能够体会到法官裁判中饱含的对群众疾苦感同身受的真心及对公序良俗原则、诚信原则的鼓励，从而满足群众对司法的期盼，赢得群众对司法的信任。"

**摘录理由**：强调精品意识的重要性，值得学习

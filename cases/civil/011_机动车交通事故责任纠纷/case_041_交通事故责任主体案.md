# 案例041：刘某1等机动车交通事故责任纠纷案

## 一、案例基本信息
- **案号**：(2018)粤民再162号
- **案由**：机动车交通事故责任纠纷
- **关键词**：交通事故责任、赔偿责任主体、停运损失、行人责任
- **难度等级**：⭐⭐⭐⭐（高）
- **审理法院**：广东省高级人民法院
- **承办法官**：谭玲

## 二、【撰写心得】
**核心观点：特别法优先适用**
> "根据特别法优先适用的原则，本案为机动车与行人之间发生的机动车交通事故责任纠纷，应当适用《道路交通安全法》第76条第1款的相关规定。"

**四个写作要点**：
1. **要充分认识裁判文书的重要性**
2. **要吃透案情发掘典型意义**：将看似普通的个案打磨成为具有典型意义的案例
3. **要加强法律适用说理论证**：准确理解法律条文，充分论证裁判结果的正当性和合理性
4. **要做到繁简得当表述精准**：实行裁判文书繁简分流

## 三、【裁判要点】
1. **赔偿责任主体的认定**：机动车与行人之间发生交通事故，赔偿责任主体只能是机动车一方及承保其交强险的保险公司
2. **过失相抵的适用**：行人一方即使有过错，也只在机动车对行人损失的赔偿责任负担上实行过失相抵，减轻机动车一方的赔偿责任
3. **停运损失的赔偿范围**：不应判决行人赔偿肇事车辆的损失，赔偿责任主体限于机动车一方，赔偿范围限于作为受害人的行人的损失
4. **人文关怀的体现**：避免行人及其家属承受交通事故造成的"再次伤害"

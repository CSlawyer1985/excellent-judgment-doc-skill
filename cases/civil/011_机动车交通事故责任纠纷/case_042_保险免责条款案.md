# 案例042：保险免责条款效力案

## 三、【裁判要点】
1. **保险免责条款效力的认定**：保险人仅提供投保人盖章的保险人格式化印制的投保人声明，不足以认定其已履行了对免责条款的明确说明义务
2. **明确说明义务的标准**：向投保人提供免责条款，对免责条款内容作出足以引起投保人注意的显著标识，并以常人能够理解的方式，对免责条款作出解释说明
3. **停业损失与误工费的择一支持**：交通事故造成家庭经营的个体工商户遭受财产损失，同时造成实际经营人遭受人身损害的，且实际经营人以及护理人员的全部收入均来源于个体工商户的营业收入，停业损失与误工费、护理费不应同时支持
4. **损失填补原则**：损失不能重复获赔

# 庭审准备检查清单

> **来源**: 《开庭:法官思维与庭审实务2024》
> **功能**: 快速检查庭审准备是否充分
> **适用对象**: 律师、法官、法律工作者
> **使用时间**: 庭审前1-3天

---

## 一、起诉准备检查清单

### 1.1 诉讼主张确定

- [ ] **六要素归纳法完成**
  - [ ] 时间（When）
  - [ ] 地点（Where）
  - [ ] 人物（Who）
  - [ ] 起因（Why）
  - [ ] 经过（What）
  - [ ] 结果（Result）

- [ ] **法律关系识别完成**
  - [ ] 关键词提取
  - [ ] 请求权基础确定
  - [ ] 法律条文检索

- [ ] **主张对象确定**
  - [ ] 主体资格审查
  - [ ] 客体明确
  - [ ] 权利义务清晰

### 1.2 诉讼请求检查

- [ ] **匹配性检查**
  - [ ] 与诉讼目的相匹配
  - [ ] 与法律关系相匹配
  - [ ] 与诉讼理由相匹配

- [ ] **精准性检查**
  - [ ] 请求事项周延（无遗漏）
  - [ ] 请求内容具体化（种类、数额、计算方法）
  - [ ] 请求事项可执行

- [ ] **明确性检查**
  - [ ] 诉讼请求明确、具体、无歧义
  - [ ] 与诉状保持一致
  - [ ] 基础法律关系明确

### 1.3 备位诉请考虑（如适用）

- [ ] **适用场景评估**
  - [ ] 同一行为多责任
  - [ ] 执行不能担忧
  - [ ] 裁判结果不确定性

- [ ] **四个误区排查**
  - [ ] 主位与备位有先后顺序
  - [ ] 属于同一法律关系
  - [ ] 存在互斥关系
  - [ ] 起诉时一并提出

### 1.4 反诉/抗辩策略

- [ ] **反诉策略**
  - [ ] 同一法律关系审查
  - [ ] 互负债务审查
  - [ ] 牵连关系审查
  - [ ] 抵消吞并效果评估

- [ ] **抗辩策略**
  - [ ] 程序抗辩准备
  - [ ] 实体抗辩准备
  - [ ] 证据支持准备

### 1.5 上诉状要素（如二审）

- [ ] 请求明确
- [ ] 事实理由重点突出
- [ ] 条理清晰
- [ ] 逻辑严密

## 三、开庭前10个问题检查

### 核心检查清单

1. **诉讼请求是否明确？** ____________
   - [ ] 是，明确具体无歧义
   - [ ] 否，需要进一步明确

2. **证据是否齐全？** ____________
   - [ ] 是，证据完整
   - [ ] 否，需要补充

3. **证人是否需要出庭？** ____________
   - [ ] 是，已安排证人出庭
   - [ ] 否，无需证人出庭

4. **鉴定是否需要申请？** ____________
   - [ ] 是，已准备鉴定申请
   - [ ] 否，无需鉴定

5. **庭审提纲是否准备？** ____________
   - [ ] 是，庭审提纲已完成
   - [ ] 否，需要准备

6. **对方可能提出什么抗辩？** ____________
   - [ ] 已预判并准备应对方案
   - [ ] 未充分预判

7. **如何应对对方的抗辩？** ____________
   - [ ] 应对方案已准备
   - [ ] 需要进一步准备

8. **调查取证是否需要申请？** ____________
   - [ ] 是，已准备调查取证申请
   - [ ] 否，无需申请

9. **和解调解是否可能？** ____________
   - [ ] 是，已准备调解方案
   - [ ] 否，不符合调解条件

10. **庭审策略是什么？** ____________
    - [ ] 庭审策略已明确
    - [ ] 需要进一步明确

**使用说明**：
1. 本清单应在庭审前1-3天开始使用
2. 逐项检查，完成的项目打勾
3. 未完成的项目应尽快补充
4. 特殊案件应根据案件类型补充特殊准备

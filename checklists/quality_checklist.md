# 裁判文书质量检查清单

## 使用说明

本检查清单用于评判裁判文书的质量，按照六个维度进行检查，每个维度10分，总分60分。

**评级标准**：
- 优秀：54-60分（90%以上）
- 良好：45-53分（75%-89%）
- 合格：36-44分（60%-74%）
- 不合格：35分以下（60%以下）

---

## 一、形式规范（10分）

### 检查项

- [ ] **标题规范**（2分）
  - [ ] 法院名称正确
  - [ ] 文书种类正确
  - [ ] 案号完整准确
  - [ ] 格式：XX法院 + 民事判决书 + （20XX）X法民初字第XX号

- [ ] **当事人信息完整**（3分）
  - [ ] 原告信息完整（姓名、性别、出生日期、民族、工作单位、住所地等）
  - [ ] 被告信息完整
  - [ ] 第三人信息完整（如有）
  - [ ] 法定代表人信息（法人或其他组织）
  - [ ] 诉讼代理人信息（如有）

- [ ] **案由准确**（2分）
  - [ ] 案由符合《民事案件案由规定》
  - [ ] 案由与争议法律关系对应

- [ ] **结构完整**（2分）
  - [ ] 首部完整
  - [ ] 正文完整
  - [ ] 尾部完整（诉讼费用、上诉权利、署名、日期、印章）

- [ ] **格式规范**（1分）
  - [ ] 字体字号符合规定
  - [ ] 段落格式规范
  - [ ] 标点符号正确

### 常见问题

❌ 当事人信息不完整（缺少住所地、出生日期等）
❌ 案由确定错误
❌ 结构残缺（缺少某一部分）
❌ 案号格式错误

### 评分标准

- 10分：完全符合规范
- 8-9分：基本符合规范，有轻微瑕疵
- 6-7分：部分不符合规范，有明显不足
- 0-5分：严重不符合规范

---

## 二、事实清楚（10分）

### 检查项

- [ ] **事实有证据支撑**（4分）
  - [ ] 所有认定的事实都有证据支持
  - [ ] 事实与证据对应关系清晰
  - [ ] 无主观臆断
  - [ ] 无无证据支撑的事实认定

- [ ] **事实表述准确**（3分）
  - [ ] 事实要素齐全（时间、地点、主体、事件、结果、原因）
  - [ ] 事实表述准确无误
  - [ ] 事实与事实的证明不混淆
  - [ ] 事实层次分明

- [ ] **事实与争议对应**（2分）
  - [ ] 事实认定围绕争议焦点展开
  - [ ] 争议事实有专门论述
  - [ ] 无争议事实简述，争议事实详述

- [ ] **事实认定达到证明标准**（1分）
  - [ ] 达到高度可能性标准（优势证据）
  - [ ] 特殊情形达到排除合理怀疑标准

### 常见问题

❌ 事实缺乏证据支撑
❌ 事实与证据矛盾
❌ 事实表述不清（要素缺失）
❌ 事实与事实的证明混淆
❌ 事实认定与争议焦点脱节

### 评分标准

- 10分：事实清楚，证据确凿
- 8-9分：事实基本清楚，偶有瑕疵
- 6-7分：事实基本成立，有明显不足
- 0-5分：事实不清或存在重大错误

---

## 三、法律正确（10分）

### 检查项

- [ ] **法律引用准确**（3分）
  - [ ] 法律名称准确完整
  - [ ] 条款序号准确
  - [ ] 内容引用准确
  - [ ] 引用格式规范

- [ ] **法律理解正确**（2分）
  - [ ] 符合立法本意
  - [ ] 符合有权解释
  - [ ] 理解无偏差

- [ ] **法律适用与事实对应**（3分）
  - [ ] 大前提（法律规范）准确
  - [ ] 小前提（案件事实）真实
  - [ ] 涵摄过程清晰
  - [ ] 事实与规范要件对应

- [ ] **法律体系完整**（2分）
  - [ ] 实体法 + 程序法
  - [ ] 法律 + 司法解释
  - [ ] 一般规定 + 特别规定
  - [ ] 无遗漏重要法律规范

### 常见问题

❌ 引用法律不准确（名称错误、条款错误）
❌ 理解法律有误
❌ 法律适用与事实脱节
❌ 遗漏重要法律规范
❌ 法律冲突处理不当

### 评分标准

- 10分：法律适用完全正确
- 8-9分：法律适用基本正确，偶有瑕疵
- 6-7分：法律适用基本成立，有明显不足
- 0-5分：法律适用错误或存在重大缺陷

---

## 四、说理充分（10分）

### 检查项

- [ ] **回应争议焦点**（3分）
  - [ ] 回应所有争议焦点
  - [ ] 每个争议焦点都有说理
  - [ ] 说理针对性强

- [ ] **证据分析充分**（2分）
  - [ ] 证据列举全面
  - [ ] 证据分析深入（三性分析）
  - [ ] 证据采信有理由
  - [ ] 综合判断有逻辑

- [ ] **法律适用说理**（3分）
  - [ ] 法律适用有理由说明
  - [ ] 法律解释有依据
  - [ ] 涵摄过程清晰

- [ ] **逻辑严密**（2分）
  - [ ] 无循环论证
  - [ ] 无推理跳跃
  - [ ] 无自相矛盾
  - [ ] 结论自然导出

### 常见问题

❌ 只给结论不给理由
❌ 理由过于简略
❌ 理由与结论脱节
❌ 回避关键争议问题
❌ 循环论证
❌ 自相矛盾

### 评分标准

- 10分：说理充分透彻，令人信服
- 8-9分：说理基本充分，偶有不足
- 6-7分：说理基本到位，有明显欠缺
- 0-5分：说理严重不足，结论突兀

---

## 五、结果明确（10分）

### 检查项

- [ ] **裁判结果具体明确**（4分）
  - [ ] 给付内容具体
  - [ ] 给付金额确定
  - [ ] 给付期限明确
  - [ ] 给付方式清晰

- [ ] **可执行性**（3分）
  - [ ] 执行标的明确
  - [ ] 执行主体明确
  - [ ] 执行条件明确
  - [ ] 执行期限明确

- [ ] **完整性**（2分）
  - [ ] 诉讼请求全覆盖
  - [ ] 反诉请求全覆盖（如有）
  - [ ] 诉讼费用明确

- [ ] **与理由对应**（1分）
  - [ ] 裁判结果与裁判理由高度对应
  - [ ] 无矛盾

### 常见问题

❌ 裁判结果不具体（如"被告应承担责任"）
❌ 给付金额不确定
❌ 给付期限不明确
❌ 超出诉讼请求裁判
❌ 遗漏诉讼请求
❌ 裁判结果与理由矛盾

### 评分标准

- 10分：裁判结果明确完整，可执行
- 8-9分：裁判结果基本明确，偶有瑕疵
- 6-7分：裁判结果基本明确，有明显不足
- 0-5分：裁判结果不明确或存在重大问题

---

## 六、语言精练（10分）

### 检查项

- [ ] **用语规范**（4分）
  - [ ] 使用法言法语
  - [ ] 用词准确
  - [ ] 无口语化
  - [ ] 无方言俚语

- [ ] **表达清晰**（3分）
  - [ ] 逻辑清晰
  - [ ] 层次分明
  - [ ] 无冗余表述
  - [ ] 无歧义表述

- [ ] **语气中立**（2分）
  - [ ] 客观中立
  - [ ] 无情感色彩
  - [ ] 无评价性语言

- [ ] **专业术语恰当**（1分）
  - [ ] 法律术语准确
  - [ ] 诉讼术语准确
  - [ ] 技术术语准确（如有）

### 常见问题

❌ 口语化严重（"签合同""打官司""还钱"）
❌ 用语不当（"恶意""无理"等贬义词）
❌ 表达冗长啰嗦
❌ 歧义表述
❌ 情绪化表达
❌ 标点符号错误

### 评分标准

- 10分：语言精练准确，文采斐然
- 8-9分：语言规范，表述清晰
- 6-7分：语言基本规范，偶有瑕疵
- 0-5分：语言混乱，表达不清

## 快速自检

使用以下问题进行快速自检：

1. 文书结构是否完整？（首部、正文、尾部）
2. 当事人信息是否完整？
3. 事实是否有证据支撑？
4. 法律引用是否准确？
5. 是否回应了所有争议焦点？
6. 裁判结果是否具体明确？
7. 是否使用了法言法语？
8. 是否存在循环论证或自相矛盾？
9. 裁判结果与理由是否对应？
10. 是否可执行？

如果以上问题全部答"是"，则文书质量基本合格。

# 说理技巧检查清单

> **版本**: v3.0
> **创建日期**: 2026年2月8日
> **来源**: 基于《双百优秀裁判文书》115个案例的说理技巧提炼
> **功能**: 帮助法官自查裁判文书说理质量，确保说理充分、严密、透彻

---

## 使用说明

**检查方式**:
- ✅ 已做到
- ⚠️ 部分做到
- ❌ 未做到

**评分标准**:
- 优秀（90-100分）：全为✅
- 良好（75-89分）：大部分✅，个别⚠️
- 合格（60-74分）：✅和⚠️各半
- 不合格（<60分）：多个❌

---

## 一、事实认定说理检查（20个技巧）

### 1.1 时间要素完整法

- [ ] 是否按时间顺序梳理事实？
- [ ] 每个时间节点是否有证据支持？
- [ ] 是否说明了时间节点之间的逻辑关系？
- [ ] 是否形成了完整的时间链条？

### 1.2 争议焦点对比法

- [ ] 是否列出了原告主张？
- [ ] 是否列出了被告抗辩？
- [ ] 是否对比分析了差异？
- [ ] 是否说明了采信理由？

### 1.3 证据链条构建法

- [ ] 是否识别了关键证据？
- [ ] 是否分析了证据之间的逻辑关系？
- [ ] 是否说明了证据如何相互印证？
- [ ] 是否形成了完整的证据链？

### 1.4 心证公开法

- [ ] 是否列出了争议证据？
- [ ] 是否分析了证据的真实性？
- [ ] 是否分析了证据的合法性？
- [ ] 是否分析了证据的关联性？
- [ ] 是否说明了综合认定结果？

### 1.5 事实要素列举法

- [ ] 是否识别了案件的关键事实要素？
- [ ] 是否逐一分析了每个要素？
- [ ] 是否说明了每个要素的认定依据？
- [ ] 是否综合形成了完整的事实认定？

### 1.6 事实推定法

- [ ] 是否确认了基础事实？
- [ ] 是否说明了推定的依据（经验法则）？
- [ ] 是否给出了对方反驳的机会？
- [ ] 是否综合认定了推定是否成立？

### 1.7 事实认定分层法

- [ ] 是否列出了无争议事实？
- [ ] 是否列出了争议事实？
- [ ] 是否对无争议事实直接认定？
- [ ] 是否对争议事实详细分析？

### 1.8 事实认定法律适用结合法

- [ ] 是否认定了事实？
- [ ] 是否引用了相关法律条文？
- [ ] 是否说明了该事实的法律意义？
- [ ] 是否直接得出了法律后果？

### 1.9 事实认定说明影响法

- [ ] 是否认定了事实？
- [ ] 是否说明了该事实的法律意义？
- [ ] 是否说明了该事实对责任认定的影响？
- [ ] 是否说明了该事实对赔偿数额的影响？

### 1.10 事实认定综合法

- [ ] 是否运用了时间要素法梳理时间线？
- [ ] 是否运用了证据链构建法形成证据链？
- [ ] 是否运用了对比法明确争议焦点？
- [ ] 是否运用了分层法区分无争议和争议事实？
- [ ] 是否综合形成了完整的事实认定？

## 二、法律适用说理检查（15个技巧）

### 2.1 涵摄方法运用

- [ ] 是否引用了完整法律条文（大前提）？
- [ ] 是否认定了案件事实（小前提）？
- [ ] 是否说明了事实如何符合法律构成要件（涵摄）？
- [ ] 是否得出了裁判结果（结论）？

### 2.2 学理辅助说理

- [ ] 是否识别了法律适用的理论争议？
- [ ] 是否梳理了不同的学术观点？
- [ ] 是否说明了选择某种观点的理由？
- [ ] 是否将理论观点适用于案件？

### 2.3 类推适用方法

- [ ] 是否识别了法律漏洞？
- [ ] 是否寻找了相似的法律规定？
- [ ] 是否说明了类推适用的理由？
- [ ] 是否注意了类推适用的限度？

### 2.4 法律解释方法运用

- [ ] 是否进行了文义解释？
- [ ] 是否进行了体系解释？
- [ ] 是否进行了历史解释？
- [ ] 是否进行了目的解释？
- [ ] 是否综合得出了解释结论？

### 2.5 法律漏洞填补方法

- [ ] 是否明确了法律漏洞的存在？
- [ ] 是否选择了填补方法？
- [ ] 是否说明了填补理由？
- [ ] 是否运用于案件？

### 2.6 法律冲突解决方法

- [ ] 是否识别了法律冲突？
- [ ] 是否确定了冲突类型？
- [ ] 是否运用了冲突规则？
- [ ] 是否确定了适用的法律？

### 2.7 法律原则运用方法

- [ ] 是否确认了缺乏具体规定？
- [ ] 是否识别了相关的法律原则？
- [ ] 是否说明了原则的适用条件？
- [ ] 是否将原则运用于案件？

### 2.8 法律习惯运用方法

- [ ] 是否确认了法律没有规定？
- [ ] 是否识别了相关的交易习惯？
- [ ] 是否证明了交易习惯的存在？
- [ ] 是否将交易习惯运用于案件？

### 2.9 外国法参考方法

- [ ] 是否识别了案件的新颖性或疑难性？
- [ ] 是否查找了相关的外国法规定？
- [ ] 是否分析了外国法的可借鉴性？
- [ ] 是否结合中国法情，谨慎参考？

### 2.10 法律适用综合方法

- [ ] 是否运用了涵摄方法进行基本论证？
- [ ] 是否运用了学理辅助增强理论深度？
- [ ] 是否运用了解释方法明确法律含义？
- [ ] 是否运用了原则方法填补漏洞？
- [ ] 是否综合得出了结论？

## 四、文书格式检查（5个技巧）

### 4.1 要素式文书"夹叙夹议"

- [ ] 是否识别了固定要素（无争议要素）？
- [ ] 是否识别了争议要素？
- [ ] 是否对固定要素简洁表述？
- [ ] 是否对争议要素详细分析？
- [ ] 是否实现了叙述与议论结合？

### 4.2 令状式文书"极简格式"

- [ ] 是否确认了案件适用令状式文书？
- [ ] 是否简化了当事人信息？
- [ ] 是否简化了案件由来？
- [ ] 是否直接给出了裁判主文？
- [ ] 是否确保了庭审录音录像留痕？

### 4.3 表格式文书"清晰明了"

- [ ] 是否确认了案件适用表格式文书？
- [ ] 是否设计了表格结构？
- [ ] 是否填写了各项信息？
- [ ] 是否确保了表格清晰易读？

### 4.4 传统式文书"详尽周密"

- [ ] 是否完整陈述了原告诉称？
- [ ] 是否完整陈述了被告辩称？
- [ ] 是否详细列举了证据？
- [ ] 是否详细分析了认证？
- [ ] 是否完整查明了事实？
- [ ] 是否详细说了理？
- [ ] 是否完整了裁判主文？

### 4.5 综合式文书"灵活运用"

- [ ] 是否分析了案件特点？
- [ ] 是否选择了合适的文书格式？
- [ ] 是否混合使用了多种格式？
- [ ] 是否确保了文书清晰易读？
- [ ] 是否实现了最佳说理效果？

### 5.2 改进建议

#### 如果整体质量为"优秀"

**继续保持**：
- ✅ 继续保持高水平说理
- ✅ 继续创新说理方法
- ✅ 继续培育精品文书
- ✅ 分享说理经验

#### 如果整体质量为"良好"

**提升方向**：
- ⚠️ 针对薄弱技巧加强
- ⚠️ 向优秀文书学习
- ⚠️ 参加培训提升能力
- ⚠️ 多写多练多反思

#### 如果整体质量为"合格"

**改进重点**：
- ❌ 系统学习说理方法
- ❌ 对照优秀文书找差距
- ❌ 请资深法官指导
- ❌ 参加专项培训

#### 如果整体质量为"不合格"

**紧急改进**：
- ❌❌ 立即停止按现有方法撰写
- ❌❌ 系统学习基础说理规范
- ❌❌ 请资深法官一对一指导
- ❌❌ 参加系统培训

### 5.3 持续改进计划

## 第一章 人身类犯罪

### 案例1：王某某故意杀人案

**【专家点评】**: 王晓东，最高人民法院刑事审判第二庭庭长，全国审判业务专家

**【评析要点】**：

1. **争议焦点总结准确，评判分析有的放矢，把握了释法说理的精度**
   - 精确表达当事人诉求，体现了对上诉权和辩护权的充分尊重
   - 将上诉理由归纳为四个方面：防卫过当、故意伤害罪认定、被害人过错、自首构成

2. **针对事实争议和法律争议采用不同说理模式，展示了辨法析理的深度**
   - **事实争议处理方法**：运用证据分析（如证人证言前后变化、医院诊断结论、被告人供述稳定性）
   - **法律争议处理方法**：先说明基本事实 → 运用法律规定、刑事政策、社会道德准则分析 → 讲明法理、说透情理
   - **正反说理技巧**：在分析自首时，既从正面归纳符合要件，又从反面分析不构成翻供

3. **将法律政策、道德规范和生活经验有机结合，体现了阐法明理的广度**
   - 对矛盾起因、发展、激化过程全面回顾
   - 结合道德规范确认精神伤害
   - 结合生活经验论证责任归属
   - 兼顾社会生活对实质合理性的需求，具有人文色彩

**【核心启示】**：
- 优秀裁判文书要做到：推理有据、繁简适当、判断正确
- 说理三要素：事理（简明扼要说明基本事实）、法理（运用法律规定分析）、情理（结合道德准则）
- 宽严相济刑事政策的准确贯彻

### 案例8：吉某1、张某1故意伤害案（共同正当防卫）

**【专家点评】**: 冯春萍，海南师范大学法学院教授

**【评析要点】**：

1. **共同正当防卫过程中，防卫过当不必然构成共同犯罪**
   - 共同犯罪以二人以上共同实施犯罪行为为前提
   - 共同防卫人在开始实施防卫行为时，应认定行为的合法性和正当性
   - 防卫过当原则上只应由防卫人个人负责，体现罪责自负原则
   - 例外情况：所有防卫人明知某一防卫人已经防卫过当仍决定加入，或在防卫进行中明知其他防卫人已经防卫过当仍继续扩大损害后果

2. **防卫过当故意伤害罪的量刑把握精准**
   - 防卫过当致人重伤：在"三年以下有期徒刑、拘役或者管制"范围内量刑
   - 防卫过当致人死亡：降低一档在"三年以上十年以下有期徒刑"范围内量刑
   - 参考案例：
     - 朱某山案：致一人死亡 → 有期徒刑七年
     - 韩某案：致一人死亡 → 有期徒刑七年
     - 王某淇案：致一人死亡、二人轻伤 → 有期徒刑六年
     - 于某案：致一人死亡、二人重伤、一人轻伤 → 有期徒刑五年
   - 本案：致一人死亡、一人重伤、一人轻伤 → 有期徒刑六年，符合罪责刑一致

**【核心启示】**：
- 共同防卫与共同犯罪的界限认定
- 正当防卫的"法不能向不法让步"精神
- 防卫过当的量刑标准统一性
- 被害人过错的认定对量刑的影响

4. 关注案件中存在的程序问题

**【核心启示】**：
- 犯罪阶层理论在司法实践中的运用
- 法益分析方法在罪名认定中的作用

### 案例19：朱某某虚开增值税专用发票案

**【专家点评】**: 王飞跃，中南林业科技大学政法学院教授

**【评析要点】**：

1. **罪名本质把握精准**
   - 虚开增值税专用发票罪的本质：致使国家税收遭受损失
   - 通过为他人虚开、为自己虚开、让他人为自己虚开、介绍他人虚开的方式
   - 在分析"行为与抵扣税款的因果关系""是否具有骗取国家税款的目的"时精准把握本质

2. **事实认定论证精细**
   - 关于朱某某不应承担法律后果的辩护：指出发票仅更改了销货单位名称和货物名称，未改变代码、号码、密码等关键信息
   - 关于戴某某不应承担法律后果的辩护：指出"未按照公对公业务要求落实开票流程规定""有过意思联络"
   - 关于是否抵扣的事实不清：指出"只要受票单位实施了向税务机关申请认证的行为，即可视为完成了抵扣"

3. **罪责大小分担精确**
   - 朱某某参与全案两笔犯罪，均从介绍虚开行为中获利，将好处费分发给他人以强化犯罪意志
   - 戴某某多次实施虚开行为，收取数十万元非法利益，作案后销毁虚开发票的记账方式逃避查处
   - 对朱某某、戴某某均应认定为主犯，承担相应刑事责任

**【撰写心得要点】**（陈健法官）：
1. **做好前期准备工作**：认真阅卷、开庭、充分听取控辩双方意见
2. **坚持规范的行文格式**：对照《法院刑事诉讼文书样式》，确保格式规范
3. **精确简练地罗列在案证据**：遵循从客观到主观、从表象到本质的规律
4. **竭尽所能写好说理部分**：
   - 明确针对什么说理，找准关键辩护观点
   - 秉持客观公正的立场说理，让证据说话
   - 熟悉说理需要的"法律工具"
   - 敢于回应案件涉及的法律难点
5. **保持裁判文书的"干净"**：秉持客观公正立场，坚守廉洁自律底线

**【核心启示】**：
- 虚开增值税专用发票罪的认定标准
- 罪责分担的精细化论证方法
- 经济犯罪案件的说理技巧
- "有几分证据说几分话"的证据裁判原则

### 案例7：马某某故意杀人案（疑罪从无的典型适用）

**【专家点评】**: 查碧然，原宁夏回族自治区高级人民法院刑事审判第三庭庭长，三级高级法官

**【评析要点】**：

1. **坚守无罪推定原则**
   - 对关键性证据进行客观全面的审查判断
   - 决定是否采信证据以及如何确定证据的证明力
   - 公安机关情况说明的效力问题：在没有相关证据印证的情况下，不能单独作为证据使用
   - 办案民警提讯录音录像：不能证明被告人承认实施了杀人行为
   - 同监室人员证言：不能排除说的是公安机关透露的指控内容

2. **坚守证据裁判原则，熟练运用证据规则**
   - 现场勘查笔录记录有血指纹两枚，但未记录是否提取
   - 鉴定意见注明血指纹和泥掌印检材是现场提取的
   - 检材来源不明、真实性存疑是有道理的
   - 现场勘查笔录只能证明现场有血指纹，但证明不了是谁所留
   - 鉴定意见虽能证明送检的血指纹系被告人所留，但不能证明检材的来源
   - 缺少中间环节的证据（检材的提取笔录和见证程序）
   - 侦查机关没有制作提取笔录，也未按规定提取玻璃板，玻璃板已丢失

3. **准确理解和把握证据标准**
   - 证人看到被害人与他人谈话的过程
   - 现场有两个血指纹，但公安机关没有提取笔录，只检测了一个指纹
   - 上述问题都未达到排除合理怀疑的刑事案件证据标准
   - 被告人供述与在案证据没有矛盾
   - 按照被告人供述，留下指纹、掌印的可能性是存在的
   - 在这种情况下，按照疑罪从无的理念宣告被告人无罪是恰当的

**【核心启示】**：
- 疑罪从无原则的实践适用
- 证据链完整性要求
- 程序正义的重要性
- 检材来源的合法性审查
- 间接证据定案的高标准

### 案例8：俎某某正当防卫宣告无罪案

**【专家点评】**: 张天虹，山西大学法学院教授，山西省法学会刑法学研究会会长

**【评析要点】**：

1. **事实认定准确**
   - 站在中立立场，客观描述案件发生、发展过程
   - 充分尊重控辩双方关于案件事实的意见
   - 对控辩双方存在分歧的部分，要么予以搁置，要么明确表明裁判者态度
   - "不法侵害的停止应具有彻底性，并在客观方面表现出明显的不可逆转性"
   - 为判断不法侵害是否停止确立了裁判规则

2. **法律适用正确**
   - 不法侵害人仅实施了一般的不法侵害行为，没有实施严重的暴力侵害行为
   - 不宜适用特殊防卫条款（刑法第20条第3款）
   - 评判防卫是否过当，应综合判断：
     - 不法侵害行为的危险程度
     - 侵害者的主观心态
     - 侵害手段、人员多少、现场客观环境与形势
     - 防卫的条件、方式、强度和后果
     - 制止不法侵害的可能性
     - 防卫行为是否为制止不法侵害所必要
   - "不能仅仅以结果判断为准，只有结果和行为均过当，才能认定防卫过当"

3. **裁判说理充分**
   - 针对检辩双方的主张以及一审裁定书的论理
   - 勇于支持正确观点，敢于及时纠正错误
   - 遵循"权利无需向不法让步"的法理
   - 确立了认定正当防卫的裁判规则："应从制止不法侵害的实际需要出发进行全面衡量，只要防卫行为在客观上有必要，防卫强度就可以大于侵害强度"
   - 从四个方面充分回应了检辩双方的疑问：
     - 俎某某的行为是否具有防卫性质
     - 防卫行为是否系针对正在进行的不法侵害
     - 防卫行为可否认定为特殊防卫
     - 防卫行为是否属于防卫过当
   - 做到了以法为据，以理服人
   - 体现司法良知，展现司法公正
   - 实现法律效果和社会效果的有机统一

**【撰写心得要点】**（王霞法官）：
1. **全面反映诉讼参与各方意见**：对抗诉书、出庭意见书、辩护词等书面材料记载的意见精准概括
2. **精心梳理证据力求精准表述**：准确、客观、中立、全面、详略得当
3. **严格根据证据认定案件事实**：证据裁判是核心原则
4. **聚焦争议注重回应公正评判**：提炼焦点问题，逐一分析论证

**【核心启示】**：
- 正当防卫制度的司法适用
- 一般防卫与特殊防卫的区分
- 防卫过当的判断标准
- "权利无需向不法让步"的司法理念
- 无罪判决书的充分说理

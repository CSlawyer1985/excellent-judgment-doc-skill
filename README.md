## 📖 项目简介

**优秀裁判文书 Skill** 是一个专业的 AI 辅助工具，旨在为法律实务工作者提供系统化的裁判文书生成指引和质量评判服务。本项目基于最高人民法院、专业法律出版社、美国联邦司法中心等权威机构的文献资料，构建了完整的裁判思维体系、方法论体系和评判标准体系。

### 核心价值

💡 **生成指引** - 提供完整的裁判思维框架和系统化的裁判方法论
🔍 **质量评判** - 六维评判体系全面评估文书质量（形式、事实、法律、说理、结果、语言）
📚 **学习工具** - 帮助法律专业学生和实务工作者学习优秀裁判文书特征
🎯 **专业水准** - 由资深律师 + AI 专家双重背景打造，兼具法律专业性和技术先进性

## 🏗️ 核心框架

### 一、裁判思维体系

```
裁判思维五要素：
1. 法律思维    - 法律发现、法律解释、法律适用
2. 逻辑思维    - 演绎推理、归纳推理、类比推理、辩证推理
3. 证据思维    - 证据审查、证明责任、证明标准
4. 价值衡量思维 - 利益识别、利益排序、利益平衡
5. 实践思维    - 经验判断、社会效果、可执行性
```

### 二、裁判方法论体系

```
四大方法论：
1. 事实认定方法论
   - 事实认定原则
   - 证据审查方法
   - 事实推理方法
   - 事实表述规范

2. 法律适用方法论
   - 法律发现方法
   - 法律解释方法
   - 法律漏洞填补
   - 法律冲突解决

3. 利益衡量方法论
   - 利益识别
   - 利益排序
   - 利益平衡

4. 裁判论证方法论
   - 演绎推理
   - 归纳推理
   - 类比推理
   - 辩证推理
```

### 三、优秀裁判文书特征体系

```
五大特征：
1. 结构特征 - 标准结构框架、章节安排逻辑、段落组织方式
2. 内容特征 - 要素完整、事实清晰、法律准确
3. 逻辑特征 - 大小前提清晰、推理链条完整、结论自然导出
4. 语言特征 - 用语规范、表达简洁、语气中立
5. 说理特征 - 针对性强、充分性够、透彻度高、说服力强
```

### 四、裁判文书评判标准体系

```
六维评分体系：
┌─────────────┬────────┬──────────┬─────────────────┐
│   评判维度   │  权重  │  满分    │    评分要点     │
├─────────────┼────────┼──────────┼─────────────────┤
│ 形式规范     │  10%   │  10/10   │ 格式、结构、要素 │
│ 事实清楚     │  20%   │  20/20   │ 证据、准确、对应│
│ 法律正确     │  25%   │  25/25   │ 引用、理解、适用│
│ 说理充分     │  25%   │  25/25   │ 回应、分析、论证│
│ 结果明确     │  10%   │  10/10   │ 具体、执行、完整│
│ 语言精练     │  10%   │  10/10   │ 规范、清晰、中立│
├─────────────┼────────┼──────────┼─────────────────┤
│   总分      │ 100%   │ 100/100  │  优秀≥90 良好≥75│
└─────────────┴────────┴──────────┴─────────────────┘

评级标准：
- 优秀（Excellent）：90-100分
- 良好（Good）：75-89分
- 合格（Qualified）：60-74分
- 不合格（Unqualified）：<60分
```

## 🚀 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/CSlawyer1985/excellent-judgment-doc-skill.git

# 进入目录
cd excellent-judgment-doc-skill
```

### 使用方式一：评判裁判文书质量

**步骤 1：准备裁判文书**

将裁判文书转换为文本格式（PDF 复制或文字版），确保文书内容完整。

**步骤 2：调用 Skill 评判**

```
请评判这份裁判文书的质量。

[粘贴裁判文书内容]

请从以下方面进行评判：
1. 形式规范
2. 事实认定
3. 法律适用
4. 说理充分性
5. 结果明确性
6. 语言表达

并给出具体评分和优化建议。
```

**步骤 3：获取评判报告**

AI 将输出：

- 📊 总体评级和分数
- 📋 六个维度的详细评分
- ✅ 具体优点（带行号引用）
- ⚠️ 具体问题（带行号引用和优化建议）
- 📝 修改示例（Before/After 对比）

### 使用方式二：生成裁判文书

**步骤 1：准备案件信息**

收集以下信息：

- 当事人基本信息（姓名、性别、出生日期、住址等）
- 案件事实（时间、地点、经过、结果）
- 争议焦点（双方争议的核心问题）
- 证据材料（证据名称、证明目的）
- 诉讼请求（具体要求）
- 法律依据（相关法律条款）

**步骤 2：调用 Skill 生成**

```
请帮我写一份[案型]的民事判决书。

当事人信息：...
案件事实：...
争议焦点：...
证据材料：...
诉讼请求：...
法律依据：...
```

**步骤 3：审核调整**

- 检查事实认定是否准确
- 核对法律引用是否正确
- 确认裁判结果是否合理
- 调整语言表达

### 使用方式三：学习裁判思维

**推荐学习路径**：

1. **入门**（30分钟）

   - 阅读 [quick_reference.md](./quick_reference.md)
   - 了解四六五五框架（四大体系、六维标准、五维思维、五大特征）
2. **进阶**（2小时）

   - 阅读 [SKILL.md](./SKILL.md)
   - 掌握核心指令和操作流程
3. **深入**（1天）

   - 阅读 [analysis_report.md](./analysis_report.md)（50,000 字完整分析报告）
   - 系统学习裁判思维、方法论、特征、标准四大体系
4. **实践**（持续）

   - 使用检查清单自检文书
   - 使用评分表评判文书
   - 使用模板练习撰写
   - 使用问题诊断表改进文书

## ❓ 常见问题

### Q1：这个 Skill 适合哪些人群使用？

**A**：

- 👨‍⚖️ **法官/法官助理**：撰写判决书、裁定书
- 👨‍💼 **律师/法律顾问**：预测裁判思路、起草文书

### Q2：AI 生成的文书可以直接使用吗？

**A**：❌ **不可以！** AI 生成的内容必须经过专业法律人士审核：

- ⚠️ 检查事实认定是否准确
- ⚠️ 核对法律引用是否正确
- ⚠️ 确认裁判结果是否合理
- ⚠️ 调整语言表达和风格

**重要提示**：使用本 Skill 产生的法律后果由使用者承担。

### Q3：评判标准是如何制定的？

**A**：评判标准基于最高人民法院、专业法律出版社、美国联邦司法中心等权威机构文献，涵盖裁判思维、方法论、文书特征和评判标准四大体系。

### Q4：评分标准是什么？

# 优秀裁判文书 Skill

## 概述

优秀裁判文书 Skill 是一个专业的AI辅助工具，用于：
1. **指引裁判文书生成**：提供系统化的裁判文书撰写规范和指引
2. **评判裁判文书质量**：对已有的裁判文书进行质量评估和优化建议

## 核心功能

### 1. 裁判文书生成指引
- 提供完整的裁判思维框架
- 给出系统化的裁判方法论
- 明确优秀裁判文书的特征标准
- 提供结构化的文书撰写模板

### 2. 裁判文书质量评判
- 形式规范检查
- 事实认定评判
- 法律适用评判
- 说理充分性评判
- 语言表达评判
- 综合评分与优化建议

## 核心理念

基于以下权威文献整理而成：
- 《最高人民法院民事审判要旨与思维》（最高人民法院民事审判第一庭）
- 《优秀民商事裁判文书标准研究》（法律出版社）
- 《法官思维的印记》（专业著作）
- 《法官裁判文书写作指南 第2版》（美国联邦司法中心）

## 使用方法

### 方式一：生成裁判文书

**步骤1：准备案件信息**
收集以下信息：
- 当事人基本信息（姓名、性别、出生日期、住址等）
- 案件事实（时间、地点、经过、结果）
- 争议焦点（双方争议的核心问题）
- 证据材料（证据名称、证明目的）
- 诉讼请求（具体要求）
- 法律依据（相关法律条款）

**步骤2：向AI提出请求**
```
请帮我写一份[案型]的民事判决书。
当事人信息：...
案件事实：...
争议焦点：...
证据材料：...
诉讼请求：...
```

**步骤3：AI生成文书**
AI将按照以下步骤生成：
1. 分析案件类型和争议焦点
2. 检索相关法律规范
3. 梳理案件事实
4. 组织证据分析
5. 进行法律适用
6. 形成裁判理由
7. 明确裁判结果
8. 组织文书结构

**步骤4：人工审核调整**
- 检查事实认定是否准确
- 核对法律引用是否正确
- 确认裁判结果是否合理
- 调整语言表达

---

### 方式二：评判裁判文书

**步骤1：准备裁判文书**
- 将裁判文书转换为文本格式（PDF复制或文字版）
- 确保文书内容完整

**步骤2：向AI提出请求**
```
请评判这份裁判文书的质量。

[粘贴裁判文书内容]

请从以下方面进行评判：
1. 形式规范
2. 事实认定
3. 法律适用
4. 说理充分性
5. 结果明确性
6. 语言表达

并给出具体评分和优化建议。
```

**步骤3：AI进行评判**
AI将：
1. 逐项检查6个维度
2. 给出每个维度的评分
3. 列出具体优点
4. 指出具体问题（标注位置）
5. 提供优化建议
6. 给出修改示例

**步骤4：根据建议优化**
- 根据AI指出的问题进行修改
- 参考AI提供的修改示例
- 优化后可再次请求AI评判

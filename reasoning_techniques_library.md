### 1.1 时间要素完整法

**案例来源**: 吉某某等和殷某1等生命权健康权身体权纠纷案

**技巧说明**:
按时间顺序梳理事实，每个时间节点都有证据支持，形成完整的时间链条

**操作步骤**:
1. 按时间顺序列出关键事件
2. 为每个时间节点标注证据来源
3. 说明时间节点之间的逻辑关系
4. 形成完整的时间链条

**模板示例**:
```markdown
本院查明：
2025年1月5日（证据1），原告与被告在XX体育中心进行篮球比赛...
2025年1月5日14:30（证据2：监控录像），双方发生身体接触，原告倒地受伤...
2025年1月5日15:00（证据3：证人证言），原告感到不适，被送往医院...
2025年1月5日16:30（证据4：医疗记录），医院诊断为骨折...

上述时间链条清晰，证据相互印证，足以认定原告受伤的时间、地点和原因。
```

**适用场景**:
- 侵权责任纠纷
- 合同履行纠纷
- 劳动争议纠纷

### 1.2 争议焦点对比法

**技巧说明**:
对比双方主张，明确争议焦点，逐一回应

**操作步骤**:

2. 列出被告抗辩

**模板示例**:

第二，被告的抗辩缺乏事实依据，因为...
第三，根据《民法典》第X条规定...

**适用场景**:
- 离婚纠纷

### 1.3 证据链条构建法

**案例来源**: 多个合同纠纷案例

**技巧说明**:
将多个证据按照逻辑关系连接，形成完整的证据链

**操作步骤**:
1. 识别关键证据
2. 分析证据之间的逻辑关系
3. 说明证据如何相互印证
4. 形成完整的证据链

**模板示例**:
```markdown
关于XX事实的认定：
证据1（合同书）证明了双方的合同关系
证据2（履行记录）证明了原告的履行行为
证据3（被告确认函）证明了被告对债务的确认
证据4（催收记录）证明了原告的催收行为

上述证据形成完整的证据链，足以证明：
1. 双方存在合法的合同关系
2. 原告已履行合同义务
3. 被告未履行付款义务
4. 原告的诉讼请求未超过诉讼时效
```

**适用场景**:
- 合同纠纷
- 债权债务纠纷
- 侵权纠纷

### 1.5 事实要素列举法

**案例来源**: 劳动争议案件系列

**技巧说明**:
将案件事实分解为若干要素，逐一认定

**操作步骤**:
1. 识别案件的关键事实要素
2. 逐一分析每个要素
3. 说明每个要素的认定依据
4. 综合形成完整的事实认定

**模板示例**:
```markdown
根据《劳动合同法》第X条的规定，判断用人单位是否应支付经济补偿金，需审查以下要素：

要素1：劳动合同解除的原因
经查，双方劳动合同于X年X月X日解除，解除原因为XXX（证据X）。

要素2：用人单位是否存在违约行为
根据XXX，被告的行为构成/不构成违约。

要素3：劳动者的工作年限
原告于X年X月X日入职，至X年X月X日离职，工作年限为X年X个月（证据X）。

要素4：工资标准
原告离职前12个月平均工资为X元（证据X）。

综上，被告应支付原告经济补偿金X元（X年×X元/年）。
```

**适用场景**:
- 劳动争议
- 社保纠纷
- 工伤赔偿

原告于X年X月X日参加股东会议并行使表决权（证据X：股东会决议）。

## 二、法律适用说理技巧（15例）

### 2.1 涵摄方法运用

**案例来源**: 多个典型案例

**技巧说明**:
运用三段论推理，将案件事实涵摄于法律条文之下

**操作步骤**:
1. 大前提：引用完整法律条文
2. 小前提：认定案件事实
3. 涵摄：说明事实如何符合法律构成要件
4. 结论：得出裁判结果

**模板示例**:
```markdown
关于XX问题的法律适用：

一、大前提（法律规范）
根据《民法典》第X条规定：
"XXX（完整引用法条）"

该条文的构成要件包括：
1. XXX
2. XXX
3. XXX

二、小前提（案件事实）
本案中，关于XX问题，本院查明：
1. XXX（事实1）
2. XXX（事实2）
3. XXX（事实3）

三、涵摄（事实符合构成要件）
将案件事实与法律条文对照分析：
1. 事实1符合构成要件1，因为XXX
2. 事实2符合构成要件2，因为XXX
3. 事实3符合构成要件3，因为XXX

因此，本案事实完全符合《民法典》第X条规定的构成要件。

四、结论（法律后果）
被告应当XXX（法律后果）。
```

**适用场景**:
- 所有案件的法律适用
- 特别是构成要件复杂的案件

五、综合解释结论

## 四、文书格式技巧（5例）

### 4.1 要素式文书"夹叙夹议"

**案例来源**: 劳动争议案件系列

**技巧说明**:
对于要素式文书，采用"夹叙夹议"的方式，固定要素一句话概括，争议要素详细分析

**操作步骤**:
1. 识别固定要素（无争议要素）
2. 识别争议要素
3. 固定要素简洁表述
4. 争议要素详细分析
5. 叙述与议论结合

**模板示例**:
```markdown
【固定要素】
一、劳动关系
原告于X年X月X日入职被告处，双方签订劳动合同，约定原告担任XX岗位，月工资X元。
上述事实双方无争议，本院予以确认。

二、工作年限
原告自X年X月X日入职至X年X月X日离职，工作年限为X年X个月。
上述事实双方无争议，本院予以确认。

【争议要素1：工资标准】
原告主张：其月工资为X元，提供银行流水为证（证据X）。
被告抗辩：原告月工资为Y元，但未提供证据。
本院认为：根据《劳动合同法》第X条规定，用人单位对工资标准负有举证责任。
被告未能提供证据证明其主张，应承担不利后果。
因此，本院采信原告的主张，认定原告月工资为X元。

【争议要素2：加班费】
原告主张：其存在加班情况，被告应支付加班费X元，提供考勤记录为证（证据X）。
被告抗辩：原告不存在加班情况，对考勤记录不予认可。
本院认为：XXX（详细分析说理）
```

**适用场景**:
- 劳动争议案件
- 其他适合要素式文书的案件

### 4.2 令状式文书"极简格式"

审判员  XXX

书记员  XXX

**适用场景**:
- 小额速裁案件
- 事实清楚、权利义务关系明确的案件
